# 分析示例

## 示例 1：技术选型

**输入**：需要在React、Vue、Angular中选择前端框架

```
Step 1: 列出候选方案
→ React、Vue、Angular、Svelte

Step 2: 评估解释力
→ 都能满足核心需求：组件化、状态管理、路由
→ React生态最丰富，Vue学习曲线最低，Angular最完整

Step 3: 量化复杂度
→ React：假设数3（需要额外选路由库、状态库、构建工具）
→ Vue：假设数2（内置路由和状态，但需选构建工具）
→ Angular：假设数1（全家桶，但需学习RxJS和TypeScript深度用法）
→ Svelte：假设数2（新生态，第三方库少）

Step 4: 应用剃刀
→ 解释力等价（都能满足需求）
→ Angular假设最少但学习成本最高
→ Vue在假设数和学习成本间最佳平衡
→ 选择Vue
```

**输出**：
```
## 分析结论

### 核心发现
1. 所有框架都能满足核心需求，解释力等价
2. Vue在复杂度和功能完整性间最佳平衡
3. React需要额外选择3个库，增加决策复杂度

### 建议
选择Vue，理由：假设数少、学习曲线平缓、生态足够成熟
```
