# 常见陷阱

## 陷阱 1：最近信息过度加权

**问题**：因最近事件更容易回忆而赋予过高权重。

❌ **Before（错误做法）**
```markdown
## 分析结论
小王最近一个月懈怠，年终评估应该降级。
建议：从优秀降到良好。
```
*问题：近因效应使1个月(8%)过度加权，忽视11个月优秀表现*

✅ **After（正确做法）**
```markdown
## 分析结论
### 全年数据加权
| 时段 | 月数 | 达标率 | 权重 | 贡献 |
|------|------|--------|------|------|
| 前11个月 | 11 | 100% | 92% | 92% |
| 最近1个月 | 1 | 70% | 8% | 5.6% |
| 全年 | 12 | - | - | 97.6% |

### 建议
1. 全年达标率97.6%，仍应评为优秀
2. 了解最近懈怠原因，而非直接降级
```
*优势：按时间加权评估，揭示全年真实表现*

**原因**：Murdock(1962)系列位置实验证明：列表末尾项目回忆率最高。最近信息更容易回忆，被误认为更重要。

## 陷阱 2：将短期波动误认为趋势

**问题**：因最近变化而推断趋势改变。

❌ **Before（错误做法）**
```markdown
## 分析结论
股票最近3天下跌，趋势不好了。
建议：赶紧卖掉。
```
*问题：近因效应使3天波动被误认为趋势反转*

✅ **After（正确做法）**
```markdown
## 分析结论
### 时间序列分析
| 周期 | 涨跌幅 | 性质 |
|------|--------|------|
| 3天 | -5% | 短期波动 |
| 1个月 | +8% | 中期向上 |
| 6个月 | +25% | 长期向上 |

### 建议
1. 3天下跌是长期向上趋势中的正常波动
2. 基本面未变，无需因短期波动卖出
```
*优势：多周期分析区分波动和趋势*

**原因**：近因效应使最近信息主导判断。多周期分析提供完整视角。

## 陷阱 3：情绪放大近因事件

**问题**：最近事件触发更强情绪反应，影响理性判断。

❌ **Before（错误做法）**
```markdown
## 分析结论
最近这次失误太严重了，他是不是不适合这个岗位？
建议：考虑调岗。
```
*问题：近因效应+情绪放大使单次失误被过度解读*

✅ **After（正确做法）**
```markdown
## 分析结论
### 失误频率分析
| 指标 | 值 |
|------|-----|
| 总任务数 | 100 |
| 失误次数 | 2 |
| 失误率 | 2% |
| 行业基准 | 5% |

### 建议
1. 失误率2%远低于行业基准5%
2. 最近失误触发的情绪反应可能放大问题
3. 基于整体表现评估，而非单次事件
```
*优势：频率分析揭示情绪反应与客观现实的差距*

**原因**：最近事件情绪反应更强，情绪进一步放大事件重要性。
