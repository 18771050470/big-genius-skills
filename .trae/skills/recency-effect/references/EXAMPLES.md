# 分析示例

## 示例 1：近因效应——员工绩效评估

**输入**：小王今年整体表现不错，但最近一个月有点懈怠，年终评估应该反映这个问题。

```
Step 1: 识别近因信号 → 产出：发现"最近一个月...应该反映"表述，最近信息获得不成比例权重
Step 2: 评估时间权重 → 产出：最近1个月(8%) vs 前11个月(92%)，但最近可能被过度加权
Step 3: 分析近因机制 → 产出：记忆可得性：最近事件更容易回忆；情绪放大：最近懈怠引发更强情绪反应
Step 4: 校正近因效应 → 产出：建议按全年数据评估，而非最近表现
```

**输出**：
```
## 分析结论
### 核心发现
1. 最近1个月懈怠获得不成比例的关注，前11个月优秀表现被忽视
2. 全年数据：11个月达标+1个月部分达标，整体达标率95%
3. 近因效应使1个月懈怠可能被放大为"态度问题"

### 建议（按优先级）
1. [高优先级] 基于全年数据评估：达标率95%，评级应为优秀
2. [高优先级] 了解最近懈怠原因：疲劳？个人问题？
3. [中优先级] 设置持续监控，而非基于最近表现决策

### 关键假设
- 前11个月数据可靠
- 最近懈怠有具体原因

### 风险
- 近因效应可能导致评估不公
- 忽视全年表现可能影响员工士气
```

## 示例 2：近因效应——投资决策

**输入**：这只股票最近连续跌了3天，趋势不好了，赶紧卖掉。

```
Step 1: 识别近因信号 → 产出：发现"最近连续跌了3天"表述，短期波动被误认为趋势
Step 2: 评估时间权重 → 产出：3天波动 vs 长期趋势，短期噪声被误认为信号
Step 3: 分析近因机制 → 产出：可得性：最近下跌更容易回忆；外推倾向：将短期趋势外推长期
Step 4: 校正近因效应 → 产出：建议分析长期趋势和基本面，而非短期波动
```

**输出**：
```
## 分析结论
### 核心发现
1. 3天下跌是短期波动，长期趋势（6个月）仍向上
2. 基本面未变：营收增长20%，利润率稳定
3. 3天下跌原因：大盘调整，非公司特定问题

### 建议（按优先级）
1. [高优先级] 基于基本面和长期趋势决策，而非3天波动
2. [高优先级] 设置止损线（如-15%），而非情绪驱动卖出
3. [中优先级] 理解短期波动是正常市场行为

### 关键假设
- 基本面数据可靠
- 大盘调整是短期现象

### 风险
- 近因效应可能导致低位卖出
- 需要区分短期波动和趋势反转
```

## 反例：不该触发此 Skill 的场景

**原因**：这是持续性问题（3天持续报错），不是单一近因事件。系统报错需要立即响应，不是近因偏差。
