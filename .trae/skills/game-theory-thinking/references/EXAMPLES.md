# 分析示例

## 示例 1：价格战决策

**输入**：竞品降价20%，我们是否跟进？

```
Step 1: 识别参与者
→ 我们、竞品A、竞品B、用户
→ 目标：我们（利润最大化）、竞品（市场份额）、用户（最低价格）

Step 2: 定义策略空间
→ 我们：降价20%、降价10%、不降价
→ 竞品A：继续降价、恢复原价、维持现价
→ 竞品B：观望、跟进降价

Step 3: 构建收益矩阵
→ 我们都降价20%：利润-30%，市场份额不变（零和）
→ 我们不降，竞品降：我们份额-15%，竞品利润-20%
→ 我们降10%：利润-10%，份额-5%

Step 4: 求解均衡
→ 纳什均衡：都降价（囚徒困境）
→ 但重复博弈下： tacit collusion（默契不降价）可能
→ 策略：降价10%+增值服务差异化
```

**输出**：
```
## 分析结论

### 核心发现
1. 完全跟进价格战是囚徒困境，双方都受损
2. 不跟进短期损失份额，但可能迫使竞品恢复理性
3. 重复博弈下，差异化比价格战更优

### 建议
1. [高] 降价10%+增值服务，平衡份额和利润
2. [中] 与竞品建立 tacit collusion 信号（通过公开声明）
3. [低] 完全跟进价格战（最差选择）
```
