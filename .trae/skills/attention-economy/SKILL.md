# 注意力经济 (Attention Economy)

## 核心定义

由诺贝尔经济学奖得主赫伯特·西蒙（Herbert A. Simon）于1971年提出的概念，指出"信息的丰富导致注意力的贫困"。在信息爆炸时代，注意力成为稀缺资源，成为数字经济中的核心货币，决定了内容、平台和商业模式的价值。

- **核心命题**：
  - 信息供给无限增长
  - 注意力供给有限
  - 注意力成为稀缺资源
  - "信息的丰富导致注意力的贫困"
- **市场特征**：
  - 零边际成本复制
  - 注意力零和博弈
  - 平台中介作用
  - 算法分发主导
- **价值转移**：
  - 从内容到注意力

## 关键原则

1. **注意力是21世纪的稀缺货币**：比信息更有价值。原因：信息可以无限复制，注意力有限且不可再生，谁掌握注意力谁就掌握价值分配权。

2. **注意力质量比数量更重要**：深度注意力比浅层浏览更有价值。原因：深度注意力带来更好的理解、记忆和行动转化，建立长期关系。

3. **注意力争夺战是零和博弈**：一个人的注意力被A获取，就不能被B获取。原因：时间有限，多任务效率低下，注意力分配是排他性的。

4. **可持续的注意力关系基于价值交换**：而非操纵。原因：操纵可能获得短期注意力，但损害信任和长期关系，价值交换建立可持续的注意力经济。
