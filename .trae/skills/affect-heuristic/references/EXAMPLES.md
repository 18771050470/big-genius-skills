# 分析示例

## 示例 1：核电项目风险评估

**输入**：这个核电项目风险太大了，一旦发生事故就是灾难，我们不应该批准。

```
Step 1: 识别情感信号 → 产出：发现"风险太大了"、"灾难"表述，恐惧情绪主导判断
Step 2: 评估情感影响 → 产出：恐惧情绪使高估风险概率、低估收益，情感替代分析
Step 3: 分析情感机制 → 产出： dread风险（不可控、致命、延迟）被高估；熟悉风险被低估
Step 4: 校正情感启发 → 产出：建议用统计数据替代情感判断，量化风险vs收益
```

**输出**：
```
## 分析结论
### 核心发现
1. "灾难"表述反映dread风险感知，非基于统计数据
2. 核电每TWh死亡率为0.07人，远低于煤炭(24.6人)和石油(18.4人)
3. 恐惧情绪使风险概率被高估约100倍（感知概率vs统计概率）

### 建议（按优先级）
1. [高优先级] 用统计数据替代情感判断：对比各能源死亡率
2. [高优先级] 量化风险vs收益：核电减排收益vs事故风险成本
3. [中优先级] 建立风险评估框架，分离情感和理性分析

### 关键假设
- 统计数据基于历史事故率
- 现代核电技术安全性高于历史数据

### 风险
- 完全忽视情感可能低估公众接受度风险
- 需要平衡统计安全和感知安全
```

## 示例 2：投资决策中的情感偏差

**输入**：这家公司CEO丑闻缠身，感觉这家公司肯定不行了，赶紧卖掉股票。

```
Step 1: 识别情感信号 → 产出：发现"感觉肯定不行"表述，情感替代基本面分析
Step 2: 评估情感影响 → 产出：负面情绪使高估公司风险，忽视财务数据
Step 3: 分析情感机制 → 产出：情感启发使负面情感扩散到整体评价（晕轮效应）
Step 4: 校正情感启发 → 产出：建议分离CEO个人和公司基本面分析
```

**输出**：
```
## 分析结论
### 核心发现
1. CEO丑闻≠公司基本面恶化，需分离个人和公司分析
2. 公司Q3财报：营收增长15%，利润率稳定，现金流健康
3. 历史数据显示CEO变更对股价影响平均-5%，6个月内恢复

### 建议（按优先级）
1. [高优先级] 基于财务数据而非情感做投资决策
2. [高优先级] 评估CEO变更对公司运营的实际影响
3. [中优先级] 设置止损线，而非情感驱动清仓

### 关键假设
- 公司财务数据真实可靠
- CEO不直接参与日常运营

### 风险
- 完全忽视情感可能低估治理风险
- 需要区分个人丑闻和公司治理问题
```
