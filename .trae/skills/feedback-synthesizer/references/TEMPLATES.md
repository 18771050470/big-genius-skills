## 反馈分类体系模板

```markdown
## 反馈分类体系

### 一级分类
1. **Bug** — 功能异常或错误
2. **功能请求** — 新功能或改进建议
3. **体验问题** — 使用困难或困惑
4. **性能问题** — 速度慢或资源占用高
5. **表扬** — 正面反馈
6. **其他** — 无法归入以上类别

### 二级标签
- **紧急度**：P0（阻断）/ P1（严重）/ P2（一般）/ P3（轻微）
- **影响范围**：全部用户 / 大部分用户 / 部分用户 / 个别用户
- **反馈渠道**：应用商店 / 客服 / 社区 / 调研 / 社交媒体
```
