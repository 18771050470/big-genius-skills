# Brand Guidelines Skill

## 核心定义

提供70+知名公司的设计规范参考，帮助应用品牌颜色、字体和视觉标准到各类产物（文档、海报、网页等）。

---

## 技能能力

- **品牌颜色应用**：根据公司设计系统应用正确的品牌色
- **字体排版规范**：应用符合品牌调性的字体组合
- **设计系统参考**：访问70+科技公司的设计规范
- **视觉一致性**：确保产物符合目标品牌的视觉标准

## 执行流程

1. **识别品牌** → 确定目标品牌或设计风格
2. **查阅规范** → 从 references/ 获取对应设计系统
3. **应用规范** → 将颜色、字体、间距等应用到产物
4. **验证一致性** → 检查是否符合品牌标准

---

## 使用示例

### 示例 1：应用 Anthropic 品牌色到海报

**场景**：为 Claude 相关产物应用品牌色

```
1. 查阅 references/claude/README.md
2. 提取主色调、辅助色、字体规范
3. 应用到 canvas-design 产物
4. 确保符合 Anthropic 品牌指南
```

### 示例 2：参考 Stripe 设计系统

**场景**：设计支付相关界面

```
1. 查阅 references/stripe/README.md

```

## 约束与限制
