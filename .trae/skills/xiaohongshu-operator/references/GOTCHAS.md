# Gotchas

> 只写 Agent 容易犯错的**非显而易见的事实**。通用知识不要写。
> 每个 Gotcha 必须包含：问题描述 + ❌ Before（错误做法） + ✅ After（正确做法） + 原因解释。

## Gotcha 1：标题过度夸张导致限流

**问题**：标题使用过度夸张的词汇（如"震惊！""绝绝子！""必看！"）会被算法判定为低质内容，导致限流。

❌ **Before（错误做法）**
```
震惊！99%的人都不知道的护肤秘密！
```

✅ **After（正确做法）**
```
 dermatologist 推荐的 3 个低成本护肤习惯
```

**原因**：2025 年小红书算法升级后，对夸张标题的识别率提升，过度夸张的标题会被降权。专业、具体的标题更容易获得推荐。

## Gotcha 2：忽视封面文字可读性

**问题**：封面文字过小、颜色与背景对比度低，导致用户无法快速获取信息，点击率下降。

❌ **Before（错误做法）**
- 封面使用浅色文字在浅色背景上
- 文字字号小于 24px
- 文字过多，超过 10 个字

✅ **After（正确做法）**
- 封面使用高对比度配色（如黑底白字、黄底黑字）
- 文字字号≥36px
- 文字控制在 6-8 个字，突出核心信息

**原因**：小红书信息流中，用户浏览速度极快，封面需要在 0.5 秒内传达核心信息。高对比度、大字号、少文字的封面更容易吸引点击。

## Gotcha 3：标签堆砌导致降权

**问题**：为了获取更多曝光，在笔记中添加大量不相关的标签，导致算法判定为垃圾内容。

❌ **Before（错误做法）**
```
#护肤 #美妆 #穿搭 #美食 #旅行 #健身 #读书 #电影 #音乐 #宠物
```

✅ **After（正确做法）**
```
#护肤 #敏感肌修复 #平价好物 #2025护肤趋势
```

**原因**：2025 年小红书规则明确，每个笔记最多使用 5 个标签，超过即降权。标签必须与内容高度相关，否则会被判定为标签滥用。

## Gotcha 4：忽视评论区运营

**问题**：发布笔记后不及时回复评论，导致互动率下降，影响算法推荐。

❌ **Before（错误做法）**
- 发布笔记后不再关注评论区
- 对用户评论不回复或敷衍回复
- 删除负面评论

✅ **After（正确做法）**
- 发布后 30 分钟内回复前 10 条评论
- 对每条评论进行个性化回复，引导二次互动
- 对负面评论进行专业、友善的回应

**原因**：评论区互动是 CES 评分的重要组成部分。及时、高质量的评论回复能显著提升互动率，进而提升算法推荐权重。

## Gotcha 5：内容同质化导致账号降权

**问题**：长期发布相似内容，导致账号被判定为低质量账号，整体权重下降。

❌ **Before（错误做法）**
- 连续 10 篇笔记都是同一类型的内容
- 使用相同的封面模板和标题公式
- 缺乏创新，跟随热点但无独特观点

✅ **After（正确做法）**
- 内容类型多样化（图文、视频、直播、合集）
- 定期更新封面风格和标题策略
- 在热点中加入独特视角和个人经验

**原因**：小红书算法鼓励内容创新，同质化内容会被降权。保持内容多样性，能提升账号整体权重和用户粘性。
