# 分析示例

## 示例 1：回归均值——员工绩效评估

**输入**：小王上个月业绩全公司第一，这个月肯定还是优秀，给他最高目标。

```
Step 1: 识别回归信号 → 产出：发现"第一...肯定还是优秀"表述，期望极端表现持续
Step 2: 评估回归趋势 → 产出：历史数据：上月第一的员工下月平均排名下降到第8名
Step 3: 分析回归机制 → 产出：极端表现=能力+运气，运气成分不可持续；随机波动使极端值趋向平均
Step 4: 校正回归均值 → 产出：建议用长期平均而非单次极端表现评估
```

**输出**：
```
## 分析结论
### 核心发现
1. 上月第一是极端表现，历史数据显示下月平均排名第8
2. 极端表现=能力+好运，运气成分不可持续
3. 给最高目标可能不现实，应基于长期平均

### 建议（按优先级）
1. [高优先级] 基于过去6个月平均绩效评估，而非单月
2. [高优先级] 设置合理目标：基于长期平均+适度增长
3. [中优先级] 理解回归均值是统计规律，非能力下降

### 关键假设
- 历史回归数据可靠
- 业绩受随机因素影响

### 风险
- 期望极端表现持续可能导致目标不现实
- 需要区分能力和运气
```

## 示例 2：回归均值——投资回报

**输入**：这只基金去年收益率80%，今年肯定还会继续涨，赶紧买入。

```
Step 1: 识别回归信号 → 产出：发现"去年80%...今年肯定继续"表述，期望极端收益持续
Step 2: 评估回归趋势 → 产出：历史数据：年收益前10%的基金，下年平均排名下降到第40%
Step 3: 分析回归机制 → 产出：极端收益=策略+市场运气，市场运气不可持续；均值回归使极端收益趋向平均
Step 4: 校正回归均值 → 产出：建议用长期平均收益评估，而非单年极端收益
```

**输出**：
```
## 分析结论
### 核心发现
1. 去年80%收益是极端表现，历史数据显示次年平均收益回归到15%
2. 极端收益部分来自市场运气，不可持续
3. 追涨可能导致高位买入

### 建议（按优先级）
1. [高优先级] 基于5年平均收益评估：15%
2. [高优先级] 理解均值回归：极端收益后通常回归
3. [中优先级] 分散投资，不追单只基金

### 关键假设
- 历史回归规律持续
- 市场有效

### 风险
- 追涨可能导致高位接盘
- 需要区分策略能力和运气
```

## 反例：不该触发此 Skill 的场景

**输入**：这只基金过去5年平均收益20%，持续稳定，值得投资。

**原因**：这是基于长期平均的稳定表现，不是极端表现后的回归。回归均值适用于极端表现后的预测。
