# 快速原型师技术交付物模板

## MVP 规划模板

```markdown
## MVP 规划

### 核心假设
- 假设：____________________
- 验证方法：____________________
- 成功标准：____________________
- 失败标准：____________________

### 功能清单
| 功能 | 优先级 | 是否包含 | 说明 |
|------|--------|----------|------|
| 功能 A | P0 | 是 | 核心功能，必须实现 |
| 功能 B | P1 | 否 | 验证后再实现 |
| 功能 C | P2 | 否 | 后续迭代 |

### 时间计划
- 第 1 周：原型构建
- 第 2 周：用户测试
- 第 3 周：数据分析 + 迭代决策

### 资源需求
- 开发人员：1 人
- 设计支持：按需
- 测试用户：5-8 人
```
