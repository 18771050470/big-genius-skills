## L1 - 思维层

> 职责：提供决策框架，指导如何分析问题
> 触发条件：复杂问题、架构决策、风险评估、长期规划

### 1.1 通用思维模型（A-G组）

| 分组 | 思维模型 | 核心功能 |
|------|---------|---------|
| **A. 底层认知基础** | 因果思维、归纳思维、演绎思维、类比思维、抽象思维、模型思维、批判性思维、创造性思维、量化思维、算法思维、元思维等 | 问题本质分析、逻辑推理、知识迁移 |
| **B. 决策与判断** | 概率思维、贝叶斯更新、机会成本、逆向思维、二阶思维、能力圈思维、沉没成本谬误、损失厌恶、确认偏误等 | 风险评估、方案对比、决策优化 |
| **C. 系统与复杂性** | 系统思维、熵增定律、正负反馈回路、临界点、涌现、反脆弱、冗余思维、复利思维等 | 全局分析、系统动态、复杂性管理 |
| **D. 学习与认知** | 费曼学习法、成长型思维、思维模型格栅、第一性原理、悖论思维、地图不是疆域等 | 知识获取、认知升级、持续学习 |
| **E. 创新与问题解决** | 水平思考、设计思维、TRIZ、六顶思考帽、5Whys、分治思维、约束思维、极简思维等 | 创新突破、问题拆解、方案生成 |
| **F. 心理学与行为** | 基本归因错误、光环效应、幸存者偏差、现状偏误、边界思维等 | 认知偏差防御、行为理解、边界管理 |
| **G. 风险与不确定性** | 预防原则、不对称风险收益等 | 风险预防、不确定性管理 |

### 1.2 编程相关思维模型（H组）

| 分组 | 思维模型 | 核心功能 |
|------|---------|---------|
| **H1. 设计原则与哲学** | DRY、SoC、组合优于继承、快速失败、防御性编程、防腐思维、契约思维等 | 代码设计、质量保障 |
| **H2. SOLID原则** | 单一职责、开闭、里氏替换、接口隔离、依赖倒置 | 面向对象设计 |
| **H3. 架构思维** | 分层架构、微服务、事件驱动、CQRS、DDD、CAP定理、康威定律等 | 系统设计、架构决策 |
| **H4. 性能与优化** | 时间/空间复杂度、缓存策略、懒加载、异步处理、批处理、向量化等 | 性能分析、优化策略 |
| **H5. 安全思维** | 最小权限、零信任、纵深防御、安全左移、威胁建模、隐私设计等 | 安全架构、风险防御 |
| **H6. 数据思维** | 范式化、索引优化、事务管理、一致性模型、流处理、数据血缘、特征工程等 | 数据架构、数据处理 |
| **H7. AI/ML思维** | 过拟合/欠拟合、偏差-方差权衡、迁移学习、强化学习、RAG、Agent思维等 | AI系统、模型应用 |
| **H8. 工程实践** | TDD、CI/CD、Git工作流、代码审查、重构、监控、SRE等 | 工程流程、团队协作 |
| **H9. 产品思维** | 用户思维、MVP、PMF、增长黑客、JTBD、Kano模型、A/B测试等 | 产品设计、市场验证 |
| **H10. 硬件思维** | 信号完整性、EMC、DFM、中断驱动、状态机、资源约束等 | 硬件设计、嵌入式开发 |

**使用场景**：
- 复杂任务前置思维分析
- 架构决策时调用
- 风险评估时调用
- 简化方案时调用
- 执行规划时调用
- 边界设计时调用
