# 数据工程师技术交付物模板

## 数据管道设计模板

```markdown
## 数据管道：[管道名称]

### 基本信息
- 数据源：
- 目标系统：
- 更新频率：
- 数据量：

### 管道流程
1. 抽取：从 [数据源] 抽取数据
2. 转换：
   - 清洗规则：
   - 关联逻辑：
   - 聚合逻辑：
3. 加载：加载到 [目标系统]

### 质量检查
| 检查项 | 规则 | 阈值 |
|--------|------|------|
| 完整性 | 空值率 | < 5% |
| 唯一性 | 主键重复 | 0 |
| 准确性 | 金额范围 | > 0 |

### 监控告警
- 失败告警：邮件/钉钉
- 延迟告警：超过 SLA 1 小时
- 质量告警：质量检查失败
```

## 数据仓库建模模板

```markdown
## 数仓模型：[模型名称]

### 分层
- ODS：贴源层，原始数据
- DWD：明细层，清洗后数据
- DWS：汇总层，轻度聚合
- ADS：应用层，业务指标

### 事实表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户 ID |
| amount | DECIMAL | 金额 |
| created_at | TIMESTAMP | 创建时间 |

### 维度表
| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | BIGINT | 主键 |
| user_name | VARCHAR | 用户名 |
| register_date | DATE | 注册日期 |
```
