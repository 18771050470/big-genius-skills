# 数据可视化工程师技术交付物模板

## 看板设计模板

```markdown
## 看板设计：[看板名称]

### 目标用户
- 角色：
- 使用场景：
- 核心需求：

### 布局设计
| 位置 | 内容 | 图表类型 | 刷新频率 |
|------|------|----------|----------|
| 顶部 | 核心 KPI | 指标卡 | 实时 |
| 左上 | 趋势图 | 折线图 | 5 分钟 |
| 右上 | 占比图 | 饼图 | 1 小时 |
| 底部 | 明细表 | 表格 | 5 分钟 |

### 交互设计
- 筛选器：时间范围、部门、产品
- 下钻：点击趋势图查看明细
- 联动：选择部门后其他图表同步过滤

### 数据需求
| 指标 | 数据源 | 更新频率 |
|------|--------|----------|
| GMV | 订单表 | 实时 |
| 订单量 | 订单表 | 实时 |
```

## 大屏设计模板

```markdown
## 大屏设计：[大屏名称]

### 基本信息
- 分辨率：3840 x 2160 (4K)
- 观看距离：3-5 米
- 使用场景：指挥中心

### 布局
```
+------------------+------------------+
|     核心指标      |     实时告警      |
|   (4 个指标卡)    |   (滚动列表)      |
+------------------+------------------+
|                                     |
|           地图/核心图表              |
|           (占据 50% 区域)            |
|                                     |
+------------------+------------------+
|     趋势图        |     排行榜        |
|   (折线图)       |   (柱状图)        |
+------------------+------------------+
```

### 视觉风格
- 背景：深色（#0a1a2a）
- 主色：科技蓝（#00d4ff）
- 辅助色：警告黄（#ffcc00）、告警红（#ff4444）
- 字体：DIN（数字）、思源黑体（中文）

### 动画效果
- 数字跳动：金额变化时数字滚动动画
- 地图脉冲：告警位置脉冲效果
- 列表滚动：告警列表自动滚动
```
