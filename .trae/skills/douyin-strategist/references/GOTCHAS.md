# Gotchas

> 只写 Agent 容易犯错的**非显而易见的事实**。通用知识不要写。
> 每个 Gotcha 必须包含：问题描述 + ❌ Before（错误做法） + ✅ After（正确做法） + 原因解释。

## Gotcha 1：前 3 秒无钩子导致完播率低

**问题**：视频前 3 秒没有抛出冲突、悬念或痛点，用户快速划走，完播率低于 40%，无法进入更大流量池。

❌ **Before（错误做法）**
```
"大家好，今天我想和大家聊聊护肤..."
```

✅ **After（正确做法）**
```
"你花了 3000 块买的精华，可能还不如这瓶 30 块的..."
```

**原因**：抖音算法将推流周期延长至 7 天，但前 3 秒留存率是决定视频能否进入更大流量池的关键。前 3 秒留存率提升 15% 可使整体完播率提高 22%。

## Gotcha 2：忽视搜索流量

**问题**：只关注推荐流量，忽视搜索流量，导致内容长尾效应不足。

❌ **Before（错误做法）**
- 标题使用宽泛词汇（如"护肤分享"）
- 不关注抖音搜索热词榜
- 视频描述空白或随意填写

✅ **After（正确做法）**
- 标题埋入长尾关键词（如"敏感肌修复完整指南 2025"）
- 定期查看抖音搜索热词榜，及时调整选题
- 视频描述详细，包含核心关键词

**原因**：抖音日均看后搜 PV 达 1.1 亿，搜索流量是稳定的长尾流量来源。优化搜索关键词能显著提升内容的长尾曝光。

## Gotcha 3：直播带货忽视人货场匹配

**问题**：直播带货时，主播人设与产品不匹配，选品策略混乱，导致转化率低。

❌ **Before（错误做法）**
- 主播形象与产品调性不符（如甜美风格主播卖工业设备）
- 选品无策略，随意上架
- 直播场景简陋，灯光昏暗

✅ **After（正确做法）**
- 主播人设与品类匹配（如美妆博主卖护肤品）
- 选品策略：引流款（低价高频）+ 利润款（中高客单）+ 形象款（品牌调性）
- 直播场景专业，背景简洁，灯光充足

**原因**：抖音电商采用"GPM+ECPM"统一指标，人货场匹配度直接影响转化率和流量获取。匹配度高的直播间，流量提升 42%、GMV 增长 33%。

## Gotcha 4：内容同质化导致账号降权

**问题**：长期发布相似内容，导致账号被判定为低质量账号，整体权重下降。

❌ **Before（错误做法）**
- 连续 10 条视频都是同一类型的内容
- 使用相同的封面模板和标题公式
- 缺乏创新，跟随热点但无独特观点

✅ **After（正确做法）**
- 内容类型多样化（剧情、知识、Vlog、直播切片）
- 定期更新封面风格和标题策略
- 在热点中加入独特视角和个人经验

**原因**：抖音算法鼓励内容创新，同质化内容会被降权。保持内容多样性，能提升账号整体权重和用户粘性。

## Gotcha 5：忽视私域导流

**问题**：只关注公域流量，忽视私域导流，导致用户无法沉淀，复购率低。

❌ **Before（错误做法）**
- 视频和直播中不引导用户添加微信/企业微信
- 无社群运营，用户流失率高
- 仅依赖平台流量，无私域沉淀

✅ **After（正确做法）**
- 视频和直播中自然引导用户添加微信/企业微信（如"加微信领取资料"）
- 建立社群，定期分享独家内容，提升用户粘性
- 通过私域运营，提升复购率和用户终身价值

**原因**：抖音公域流量成本高，私域流量是长期价值的核心。通过"短视频引流+直播间深度互动+私域沉淀"形成闭环，能显著提升用户终身价值。
