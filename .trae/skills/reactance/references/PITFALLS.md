# 常见陷阱

## 陷阱 1：强制变更触发抗拒

**问题**：用强制手段推行变更，忽视用户自主权。

❌ **Before（错误做法）**
```markdown
## 分析结论
强制所有用户使用新版，旧版直接下线。
建议：立即执行。
```
*问题：心理抗拒使强制变更触发抵制*

✅ **After（正确做法）**
```markdown
## 分析结论
### 抗拒风险评估
| 维度 | 强制方案 | 渐进方案 |
|------|---------|---------|
| 用户选择权 | 无 | 有 |
| 预期抗拒 | 高 | 低 |
| 流失风险 | 30% | 5% |

### 建议
1. 保留选择权降低抗拒
2. 渐进过渡而非强制
```
*优势：选择权降低抗拒*

**原因**：Brehm(1966)：当自由被威胁时，人们产生心理抗拒，可能采取反向行为恢复自由。

## 陷阱 2：反向行为

**问题**：强制导致与预期相反的结果。

❌ **Before（错误做法）**
```markdown
## 分析结论
规定必须9点到，迟到扣工资。
建议：严格执行。
```
*问题：心理抗拒使员工消极抵抗*

✅ **After（正确做法）**
```markdown
## 分析结论
### 抗拒表现预测
| 行为 | 强制政策 | 弹性政策 |
|------|---------|---------|
| 准时率 | 表面高，实际低效 | 自主管理 |
| 满意度 | 低 | 高 |
| 离职率 | 高 | 低 |

### 建议
1. 强制可能触发消极抵抗
2. 弹性政策更有效
```
*优势：预测并避免反向行为*

**原因**：心理抗拒使人们采取与要求相反的行为来恢复自由感。

## 陷阱 3：过度控制

**问题**：控制越强，抗拒越大。

❌ **Before（错误做法）**
```markdown
## 分析结论
用户不按指引操作，增加更多限制。
建议：加强控制。
```
*问题：心理抗拒使控制触发更强抗拒*

✅ **After（正确做法）**
```markdown
## 分析结论
### 控制vs抗拒关系
| 控制程度 | 用户抗拒 | 效果 |
|---------|---------|------|
| 低 | 低 | 好 |
| 中 | 中 | 中 |
| 高 | 高 | 差 |

### 建议
1. 增加控制可能适得其反
2. 用引导替代限制
```
*优势：理解控制与抗拒的反比关系*

**原因**：Worchel等(1998)：控制程度与抗拒强度正相关。适度引导比强制更有效。
