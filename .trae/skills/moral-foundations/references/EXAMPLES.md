# 分析示例

## 示例 1：道德基础理论——产品功能争议

**输入**：这个功能应该加入公平性检查，确保不同用户群体获得相同待遇，这是基本道德。

```
Step 1: 识别道德基础 → 产出：发现"公平性...基本道德"表述，触发公平/欺骗道德基础
Step 2: 评估多基础冲突 → 产出：公平基础(平等对待) vs 自由基础(不干预市场)，不同人优先级不同
Step 3: 分析道德机制 → 产出：Haidt(2012)6大道德基础：关爱/伤害、公平/欺骗、忠诚/背叛、权威/颠覆、圣洁/堕落、自由/压迫
Step 4: 校正道德冲突 → 产出：建议识别各方道德基础，寻找平衡点
```

**输出**：
```
## 分析结论
### 核心发现
1. "公平性"触发公平/欺骗道德基础，但可能与其他基础冲突
2. 利益相关方道德基础差异：用户(公平)、业务(自由)、技术(圣洁-代码纯洁)
3. 道德判断直觉先于推理，不同基础导致不同结论

### 建议（按优先级）
1. [高优先级] 识别各方道德基础，理解分歧根源
2. [高优先级] 寻找多基础平衡方案：公平检查+最小干预
3. [中优先级] 用数据辅助道德讨论，而非纯直觉

### 关键假设
- 道德基础差异是分歧主因
- 平衡方案可找到

### 风险
- 道德冲突可能升级为价值观冲突
- 需要区分道德偏好和技术判断
```

## 示例 2：道德基础理论——团队管理冲突

**输入**：员工应该忠于公司，跳槽就是不忠诚，这种人不值得信任。

```
Step 1: 识别道德基础 → 产出：发现"忠于公司...不忠诚"表述，触发忠诚/背叛道德基础
Step 2: 评估多基础冲突 → 产出：忠诚基础(员工忠於公司) vs 关爱基础(员工职业发展) vs 自由基础(择业自由)
Step 3: 分析道德机制 → 产出：不同文化背景道德基础权重不同：集体主义重忠诚，个人主义重自由
Step 4: 校正道德冲突 → 产出：建议理解多元道德基础，避免单一道德判断
```

**输出**：
```
## 分析结论
### 核心发现
1. "忠诚"是忠诚/背叛道德基础，但员工也有自由/关爱基础需求
2. 跳槽可能出于职业发展(关爱)、更好机会(自由)，非背叛
3. 道德判断受文化背景影响：集体主义vs个人主义

### 建议（按优先级）
1. [高优先级] 理解员工跳槽的多重道德基础
2. [高优先级] 建立留人机制：职业发展、薪酬竞争力
3. [中优先级] 接受合理流动，非道德问题

### 关键假设
- 员工有职业发展权利
- 合理流动是正常的

### 风险
- 单一道德判断可能忽视员工合理需求
- 需要平衡公司利益和员工权益
```
