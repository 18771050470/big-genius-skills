# 常见陷阱

## 陷阱 1：单一道德基础判断

**问题**：用自己的道德基础评判他人行为。

❌ **Before（错误做法）**
```markdown
## 分析结论
员工跳槽就是不忠诚，不值得信任。
建议：不录用跳槽频繁的人。
```
*问题：仅用忠诚基础判断，忽视自由和关爱基础*

✅ **After（正确做法）**
```markdown
## 分析结论
### 多道德基础分析
| 道德基础 | 视角 | 结论 |
|---------|------|------|
| 忠诚/背叛 | 公司视角 | 跳槽=背叛 |
| 自由/压迫 | 员工视角 | 跳槽=自由 |
| 关爱/伤害 | 家庭视角 | 跳槽=更好生活 |

### 建议
1. 不同道德基础导致不同判断
2. 接受合理流动，非道德问题
```
*优势：多基础分析揭示判断差异根源*

**原因**：Haidt(2012)证明：道德判断基于直觉，不同人激活不同道德基础。政治倾向影响基础权重。

## 陷阱 2：道德化技术问题

**问题**：将技术选择上升为道德问题。

❌ **Before（错误做法）**
```markdown
## 分析结论
不用这个框架就是不专业，道德上有问题。
建议：强制使用。
```
*问题：技术选择被道德化，引发不必要冲突*

✅ **After（正确做法）**
```markdown
## 分析结论
### 技术vs道德
| 维度 | 评估 |
|------|------|
| 技术选择 | 多方案可行 |
| 道德层面 | 无道德含义 |
| 最佳实践 | 因场景而异 |

### 建议
1. 技术选择基于场景，非道德
2. 避免将偏好道德化
```
*优势：区分技术判断和道德判断*

**原因**：道德化使技术讨论升级为价值观冲突，阻碍理性决策。

## 陷阱 3：忽视文化差异

**问题**：用自己的文化道德标准评判他人。

❌ **Before（错误做法）**
```markdown
## 分析结论
他们不重视权威，这是道德问题。
建议：纠正。
```
*问题：忽视文化差异导致的道德基础权重不同*

✅ **After（正确做法）**
```markdown
## 分析结论
### 文化道德差异
| 文化 | 重视基础 | 轻视基础 |
|------|---------|---------|
| 集体主义 | 忠诚、权威 | 自由 |
| 个人主义 | 自由、公平 | 忠诚、权威 |

### 建议
1. 文化差异导致道德基础权重不同
2. 理解而非纠正差异
```
*优势：文化视角理解道德差异*

**原因**：Graham等(2011)证明：不同文化和政治倾向的人道德基础权重显著不同。
