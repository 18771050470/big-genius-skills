## 交付物模板 2：WCAG 检查清单

```markdown
# WCAG 2.2 AA 检查清单

## 1. 可感知性

### 1.1 文本替代
- [ ] 所有图片有适当的 alt 文本
- [ ] 装饰性图片 alt 为空
- [ ] 复杂图表有详细描述

### 1.2 时序媒体
- [ ] 视频有字幕
- [ ] 视频有音频描述或替代文本

### 1.3 适应性
- [ ] 内容不依赖颜色传达
- [ ] 表单标签正确关联
- [ ] 表格有标题和表头

### 1.4 可区分性
- [ ] 文本对比度 >= 4.5:1
- [ ] 大文本对比度 >= 3:1
- [ ] 页面可缩放到 200%
- [ ] 文本间距可调整

## 2. 可操作性

### 2.1 键盘可访问
- [ ] 所有功能可用键盘操作
- [ ] 无键盘陷阱
- [ ] 有跳过链接

### 2.2 足够时间
- [ ] 用户可延长时间限制
- [ ] 自动滚动可暂停

### 2.4 导航
- [ ] 页面有标题
- [ ] 焦点顺序合理
- [ ] 焦点可见
- [ ] 链接目的明确

## 3. 可理解性

### 3.1 可读性
- [ ] 页面语言已声明
- [ ] 专业术语有解释

### 3.2 可预测性
- [ ] 导航一致
- [ ] 表单提交不自动跳转

### 3.3 输入辅助
- [ ] 错误可识别
- [ ] 有错误建议
- [ ] 有防错机制

## 4. 健壮性

### 4.1 兼容性
- [ ] HTML 有效
- [ ] ARIA 使用正确
```

## 交付物模板 3：无障碍修复指南

```markdown
# 无障碍修复指南：[问题类型]

## 问题：表单标签未关联

### 影响
- 屏幕阅读器用户无法知道输入框用途
- 点击标签无法聚焦输入框

### 错误示例
```html
<!-- 错误：标签和输入框无关联 -->
<span>邮箱</span>
<input type="email">
```

### 正确示例
```html
<!-- 方法 1：隐式关联 -->
<label>
  邮箱
  <input type="email">
</label>

<!-- 方法 2：显式关联（推荐） -->
<label for="email">邮箱</label>
<input type="email" id="email">

<!-- 方法 3：ARIA 关联（当无法使用 label 时） -->
<span id="email-label">邮箱</span>
<input type="email" aria-labelledby="email-label">
```

### 验证方法
1. 使用 axe 扫描，确认无 "Form elements must have labels" 错误
2. 使用屏幕阅读器（NVDA）测试，确认标签被正确播报
3. 点击标签，确认输入框获得焦点

### 参考
- WCAG 1.3.1: Info and Relationships
- WCAG 3.3.2: Labels or Instructions
```
