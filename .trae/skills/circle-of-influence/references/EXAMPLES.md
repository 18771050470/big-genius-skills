# 分析示例

## 示例 1：关注圈与影响圈——创业焦虑

**输入**：我很焦虑，担心经济下行影响融资，担心竞品抄袭我们的功能，担心政策变化。

```
Step 1: 识别关注圈 → 产出：发现"担心经济下行...担心竞品...担心政策"表述，全部是外部不可控因素
Step 2: 识别影响圈 → 产出：可控事项：产品迭代速度、用户关系、团队能力、成本控制
Step 3: 分析精力分配 → 产出：当前精力100%在关注圈（焦虑外部因素），0%在影响圈（实际行动）
Step 4: 制定聚焦策略 → 产出：建议将焦虑转化为影响圈行动
```

**输出**：
```
## 分析结论
### 核心发现
1. 当前焦虑全部集中在不可控事项（经济、竞品、政策）
2. 影响圈内的高价值行动被忽视：产品迭代、用户留存、成本优化
3. 焦虑不产生实际价值，行动才能扩大影响圈

### 建议（按优先级）
1. [高优先级] 列出影响圈行动清单：本周可完成的3件事
2. [高优先级] 设定信息过滤规则：每天只看1次行业新闻
3. [中优先级] 建立周复盘：评估影响圈扩展进度

### 关键假设
- 焦虑源于关注圈过度投入
- 影响圈行动可缓解焦虑

### 风险
- 完全忽视关注圈可能错过重要信号
- 需要平衡关注和行动

### 下一步
建议调用 [constraint-thinking] 在影响圈内利用约束创新
```

## 示例 2：关注圈与影响圈——团队管理

**输入**：团队成员积极性不高，我担心他们离职，但不知道怎么办。

```
Step 1: 识别关注圈 → 产出：发现"担心离职"表述，他人行为不可直接控制
Step 2: 识别影响圈 → 产出：可控事项：工作环境、激励机制、沟通频率、成长机会
Step 3: 分析精力分配 → 产出：担忧他人行为（不可控）而非改善环境（可控）
Step 4: 制定聚焦策略 → 产出：建议聚焦改善团队环境和激励机制
```

**输出**：
```
## 分析结论
### 核心发现
1. 团队成员行为不可直接控制，但工作环境可改善
2. 影响圈行动：1对1沟通、成长路径、认可机制、工作自主性
3. 担忧离职是关注圈，改善环境是影响圈

### 建议（按优先级）
1. [高优先级] 每周1对1沟通，了解成员需求和痛点
2. [高优先级] 建立成长路径：技能培训、项目挑战、晋升通道
3. [中优先级] 改善工作环境：减少会议、增加自主性

### 关键假设
- 环境影响离职率
- 沟通可识别真实需求

### 风险
- 改善环境需要时间，短期可能无效
- 需要持续投入

### 下一步
建议调用 [opportunity-cost] 评估管理精力分配
```
