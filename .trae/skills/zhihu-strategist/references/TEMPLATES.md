# 知乎运营模板

## 内容日历模板

```markdown
### 月度内容日历

| 日期 | 内容类型 | 主题 | 形式 | 目标 |
|------|---------|------|------|------|
| 周一 | 热点回答 | [主题] | 回答 | 提升曝光 |
| 周三 | 深度文章 | [主题] | 文章 | 提升专业形象 |
| 周五 | 视频内容 | [主题] | 视频 | 提升互动 |
| 周日 | Live预告 | [主题] | 动态 | 提升关注 |

### 关键节点
- [节日/活动]：[内容计划]
```

## 回答结构模板

```markdown
### 回答主题：[主题]

**开门见山（1-2句）**
- 直接给出核心观点或结论

**逻辑论证（3-5段）**
- 论点1：[论据+案例]
- 论点2：[论据+案例]
- 论点3：[论据+案例]

**案例支撑（1-2个）**
- 案例1：[具体案例+分析]
- 案例2：[具体案例+分析]

**总结升华（1段）**
- 总结核心观点
- 给出行动建议或思考方向

**互动引导（1句）**
- 引导点赞、收藏、评论
```

## Live策划模板

```markdown
### Live主题：[主题]

- [受众描述]

**互动设计**
- 问答环节：[时间安排]
- 抽奖环节：[奖品+规则]

**推广计划**
- 预热时间：[时间]
- 推广渠道：[渠道]
- 预期参与人数：[人数]
```

## 数据复盘模板

```markdown
### 月度数据复盘

**核心指标**

- 分析：爆款原因/低效原因

**优化方案**
- [优化点1]
- [优化点2]
```
