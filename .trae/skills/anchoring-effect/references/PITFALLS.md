# 常见陷阱

## 陷阱 1：过度调整——为了逃避锚点而推出极端值

### 问题描述
在意识到锚定效应后，一些人会刻意远离初始锚点，导致调整过度。例如，卖方报价 $100，买方为了不被锚定，还价 $10，结果损失了可能的交易区间。

### ❌ 反例

```
场景：二手笔记本电脑交易
卖家标价：$800（锚点）
买家心理：我不能被这个锚点控制，于是直接报价 $200。
结果：卖家觉得侮辱性，拒绝谈判，交易告吹。
问题：$200 显然低于合理区间（$400-$600），过度调整导致失效。
```

### ✅ 正例

```
场景：同一交易，卖家标价 $800
买家识别锚点后，去查了同款二手成交历史（$500-$600）、配置折旧、市场供求。
建立中立范围：$480（最低配折旧）- $580（高配良好状态）
出价：$500（比中立范围下限略高，留谈判空间）
目标成交：$530
结果：卖家同意 $530，双方满意。
原因：调整基于数据，而非情绪化远离锚点。
```

### 原因分析

- **心理反弹**：人天生有“反锚定”冲动，但脱离数据支撑的调整本身也是一种偏差。
- **解决方案**：始终以独立参考源构建中立范围，而不是凭感觉“比锚点低很多”。

## 陷阱 2：忽略内在锚点——自己设定的目标也会固化判断

### 问题描述

锚定不仅来自外部，也来自我们自身设定的初始目标或预期。例如，一个人设定“我今年赚 $50,000 就满意了”（内在锚点），后续即使市场数据显示可以更高，也很难调整。

### ❌ 反例

```
场景：自由职业者谈新项目报价
内在锚点：心里默念“这项目最多 $5,000”（来自之前类似项目的经验）
甲方问：“你的报价是多少？”
因为内在锚点，直接脱口而出 $5,000。
甲方立刻同意（说明报价偏低）。
事后知道甲方预算上限是 $8,000，自己损失了 $3,000。
问题：内在锚点隐藏在思维中，没有被识别为需要打破的锚点。
```

### ✅ 正例

```
场景同一自由职业者
在接新项目前，执行锚定检查：
- 识别内在锚点：$5,000（上一次类似项目）
- 寻找独立参考：
  - 同水平设计师时薪 × 预计工时 = $6,500
  - 行业平均项目报价 = $7,000-$9,000
  - 甲方公司历史项目付款记录 = 通常支付 $7,000-$8,000
- 构建中立范围：$6,500 - $8,000
- 报价策略：首轮报 $8,000（中立范围上限，留谈判空间）
- 结果：甲方砍到 $7,200，比自己原来的 $5,000 高出 44%
原因：识别并打破了内在锚点，用独立数据重新定位。
```

### 原因分析

- **自我锚定**：人们习惯用过去的成功或失败作为基准，但环境和市场已变。
- **解决方法**：在决策前刻意列出所有可能的内在锚点（已有经验、朋友建议、个人预期），并主动质疑：“如果我没有这个过去，我会怎么判断？”
