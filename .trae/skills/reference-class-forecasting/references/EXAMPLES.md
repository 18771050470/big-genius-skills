# 分析示例

## 示例 1：参照类预测——项目时间估算

**输入**：这个AI项目我们估计需要6个月完成，团队技术能力强。

```
Step 1: 识别内部视角 → 产出：发现"6个月完成"表述，基于内部计划而非外部数据
Step 2: 寻找参照类 → 产出：类似AI项目历史数据：平均耗时9个月，范围6-14个月
Step 3: 调整预测 → 产出：基于参照类中位数9个月，根据当前项目特点调整±1个月
Step 4: 输出预测 → 产出：8-10个月，而非内部估计的6个月
```

**输出**：
```
## 分析结论
### 核心发现
1. 内部估计6个月，参照类中位数9个月，低估33%
2. 参照类数据：类似项目10个，平均9个月，范围6-14个月
3. 当前项目特点：团队强(-1月)、新技术(+1月)，净调整0

### 建议（按优先级）
1. [高优先级] 基于参照类预测：8-10个月
2. [高优先级] 设置月度里程碑，监控偏差
3. [中优先级] 识别最可能延期的环节

### 关键假设
- 参照类项目与当前可比
- 历史数据可靠

### 风险
- 内部视角导致乐观估计
- 需要外部视角校准
```

## 示例 2：参照类预测——餐厅开业预算

**输入**：开这家餐厅预算200万就够了，我们做了详细计划。

```
Step 1: 识别内部视角 → 产出：发现"200万就够了"表述，基于内部计划而非行业数据
Step 2: 寻找参照类 → 产出：同类餐厅开业数据：平均预算350万，范围200-600万
Step 3: 调整预测 → 产出：基于参照类中位数350万，根据当前情况调整
Step 4: 输出预测 → 产出：300-400万，而非内部估计的200万
```

**输出**：
```
## 分析结论
### 核心发现
1. 内部估计200万，参照类中位数350万，低估43%
2. 参照类数据：同类餐厅20家，平均350万，范围200-600万
3. 常见超支项：装修(+30%)、设备(+20%)、执照审批(+10%)

### 建议（按优先级）
1. [高优先级] 基于参照类预测：300-400万
2. [高优先级] 预留20%应急预算
3. [中优先级] 分阶段投入，降低风险

### 关键假设
- 参照类餐厅与当前可比
- 地区成本差异可调整

### 风险
- 内部视角导致预算不足
- 需要外部视角校准
```

## 反例：不该触发此 Skill 的场景

**原因**：这已经使用了参照类预测（过去5个项目数据），不需要再触发此Skill。参照类预测适用于仅用内部视角估计的场景。
