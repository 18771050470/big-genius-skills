# 常见陷阱

## 陷阱 1：处置效应

**问题**：过早卖出盈利，过晚卖出亏损。

❌ **Before（错误做法）**
```markdown
## 分析结论
亏了20%不卖，等回本。
建议：继续持有。
```
*问题：后悔厌恶使实现损失被避免*

✅ **After（正确做法）**
```markdown
## 分析结论
### 处置效应数据
| 行为 | 平均结果 | 说明 |
|------|---------|------|
| 持有亏损 | 多亏15% | 处置效应 |
| 及时止损 | 少亏5% | 理性 |

### 建议
1. 沉没成本不应影响决策
2. 基于当前价值决定持有或卖出
```
*优势：数据揭示处置效应成本*

**原因**：Odean(1998)证明：投资者卖出盈利股票的倾向比卖出亏损股票高50%。实现损失触发后悔情绪。

## 陷阱 2：沉没成本谬误

**问题**：因已投入成本而继续持有。

❌ **Before（错误做法）**
```markdown
## 分析结论
已经亏了这么多，现在卖太可惜了。
建议：继续持有。
```
*问题：后悔厌恶使沉没成本影响决策*

✅ **After（正确做法）**
```markdown
## 分析结论
### 沉没成本分析
| 成本类型 | 是否影响决策 |
|---------|------------|
| 买入成本(已发生) | 否(沉没) |
| 当前价值 | 是 |
| 未来预期 | 是 |

### 建议
1. 沉没成本不应影响决策
2. 问题应是：如果现在没持有，会买吗？
```
*优势：区分沉没成本和相关成本*

**原因**：沉没成本是已发生不可回收的成本，理性决策只考虑未来成本和收益。

## 陷阱 3：高估长期后悔

**问题**：高估不做某事的长期后悔强度。

❌ **Before（错误做法）**
```markdown
## 分析结论
不敢创业，失败会后悔一辈子。
建议：放弃。
```
*问题：后悔厌恶使高估失败后悔*

✅ **After（正确做法）**
```markdown
## 分析结论
### 后悔时间衰减
| 时间 | 行动后悔 | 遗漏后悔 |
|------|---------|---------|
| 短期 | 强 | 弱 |
| 长期 | 弱 | 强 |

### 建议
1. 长期来看，不做比做了更后悔
2. 降低风险后尝试，而非完全放弃
```
*优势：时间视角评估后悔*

**原因**：Gilovich等(1994)：长期遗憾中，遗漏(没做)比行动(做了)更强烈。
