# 分析示例

## 示例 1：后悔厌恶——投资决策

**输入**：这只股票我亏了20%了，现在卖就真亏了，再等等看能不能回本。

```
Step 1: 识别后悔信号 → 产出：发现"现在卖就真亏了...再等等"表述，因害怕实现损失而持有亏损
Step 2: 评估后悔影响 → 产出：实现损失会触发后悔情绪，使继续持有而非理性止损
Step 3: 分析后悔机制 → 产出：处置效应：过早卖出盈利、过晚卖出亏损；后悔预期：实现损失比未实现损失更痛苦
Step 4: 校正后悔厌恶 → 产出：建议基于当前价值决策，而非买入成本
```

**输出**：
```
## 分析结论
### 核心发现
1. 亏损20%是沉没成本，不应影响当前决策
2. 后悔厌恶使"实现损失"被感知为比"未实现损失"更痛苦
3. 处置效应：持有亏损股票平均比卖出盈利股票多亏损15%

### 建议（按优先级）
1. [高优先级] 基于当前价值决策：如果现在不会买，就应该卖
2. [高优先级] 设置止损规则，避免情绪决策
3. [中优先级] 理解沉没成本不应影响未来决策

### 关键假设
- 当前价值反映未来预期
- 止损规则可执行

### 风险
- 后悔厌恶可能导致亏损扩大
- 需要规则约束情绪
```

## 示例 2：后悔厌恶——职业选择

**输入**：我不敢辞职创业，万一失败了会后悔一辈子，还是稳定工作安全。

```
Step 1: 识别后悔信号 → 产出：发现"万一失败会后悔一辈子"表述，高估失败后悔强度
Step 2: 评估后悔影响 → 产出：后悔厌恶使过度规避风险，可能错过机会
Step 3: 分析后悔机制 → 产出：行动后悔(做了后悔)比遗漏后悔(没做后悔)短期更强，但长期遗漏后悔更强
Step 4: 校正后悔厌恶 → 产出：建议评估长期遗憾，而非短期后悔恐惧
```

**输出**：
```
## 分析结论
### 核心发现
1. 后悔厌恶使高估失败后悔，低估错过机会的长期遗憾
2. Gilovich & Medvec(1994)：短期行动后悔更强，长期遗漏后悔更强
3. 稳定工作的"安全"可能伴随长期职业遗憾

### 建议（按优先级）
1. [高优先级] 评估长期遗憾：不做vs做了失败
2. [高优先级] 降低创业风险：MVP验证、兼职创业
3. [中优先级] 理解后悔情绪会随时间衰减

### 关键假设
- 长期遗憾比短期后悔更重要
- 风险可降低

### 风险
- 后悔厌恶可能导致人生遗憾
- 需要平衡风险和机会
```
