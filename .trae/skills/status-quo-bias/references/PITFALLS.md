# 常见陷阱

## 陷阱 1：损失厌恶维持现状

**问题**：因迁移成本(确定损失)而放弃长期收益。

❌ **Before（错误做法）**
```markdown
## 分析结论
迁移成本太高，还是继续用老的吧。
建议：不迁移。
```
*问题：现状偏差使确定损失被过度加权*

✅ **After（正确做法）**
```markdown
## 分析结论
### 成本收益分析
| 维度 | 值 |
|------|-----|
| 迁移成本 | 2个月(确定) |
| 年收益 | 3个月/年 |
| 回本周期 | 8个月 |
| 3年净收益 | 7个月 |

### 建议
1. 8个月回本，长期收益明确
2. 分阶段迁移降低风险
```
*优势：长期ROI评估替代短期损失恐惧*

**原因**：Samuelson & Zeckhauser(1988)：即使改变有明确收益，人们仍倾向维持现状。损失厌恶使确定成本被过度加权。

## 陷阱 2：默认选项力量

**问题**：默认选项被不加批判地接受。

❌ **Before（错误做法）**
```markdown
## 分析结论
默认配置就行，用户一般不会改。
建议：保持默认。
```
*问题：现状偏差使默认选项成为事实标准*

✅ **After（正确做法）**
```markdown
## 分析结论
### 默认选项影响
| 指标 | 值 |
|------|-----|
| 接受默认率 | 80%+ |
| 用户最佳利益 | 需验证 |
| 更改难度 | 低 |

### 建议
1. 默认选项应基于用户利益设计
2. 定期评估默认是否仍最优
```
*优势：审慎设计默认选项*

**原因**：Johnson & Goldstein(2003)：器官捐献默认选项从"选择加入"改为"选择退出"，参与率从15%升到90%。

## 陷阱 3：沉没成本强化现状

**问题**：因已投入成本而继续维持现状。

❌ **Before（错误做法）**
```markdown
## 分析结论
我们在这个框架上投入了3年，不能轻易换。
建议：继续使用。
```
*问题：现状偏差+沉没成本使维持现状*

✅ **After（正确做法）**
```markdown
## 分析结论
### 沉没成本分析
| 成本 | 是否相关 |
|------|---------|
| 已投入3年 | 否(沉没) |
| 未来收益 | 是 |
| 未来成本 | 是 |

### 建议
1. 沉没成本不应影响决策
2. 基于未来收益评估
```
*优势：区分沉没成本和未来成本*

**原因**：沉没成本是已发生不可回收的成本，现状偏差使其被用来合理化维持现状。
