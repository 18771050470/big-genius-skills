# 常见陷阱

## 陷阱 1：将行为归因于个人特质

**问题**：因他人行为而推断其性格或能力，忽视情境因素。

❌ **Before（错误做法）**
```markdown
## 分析结论
小李业绩不达标，肯定是他不够努力。
建议：加强考核，不行就换人。
```
*问题：基本归因错误使高估个人因素，忽视情境因素*

✅ **After（正确做法）**
```markdown
## 分析结论
### 归因分析
| 因素类型 | 具体因素 | 影响程度 |
|---------|---------|---------|
| 个人因素 | 努力程度、能力 | 待评估 |
| 情境因素 | 市场下滑15%、竞品降价30% | 高 |
| 系统因素 | 资源支持、培训时间 | 中 |

### 建议
1. 先分析情境和系统因素
2. 与小李沟通了解具体困难
3. 历史业绩显示过去6个月平均达标率110%
```
*优势：多因素归因，避免单一特质归因*

**原因**：Ross(1977)定义基本归因错误：解释他人行为时高估个人因素、低估情境因素。Jones & Harris(1967)实验证明即使知道行为是情境强制的，仍归因于个人态度。

## 陷阱 2：对自己和他人使用不同归因标准

**问题**：自己失败归情境，他人失败归个人。

❌ **Before（错误做法）**
```markdown
## 分析结论
我业绩不好是因为市场差，他业绩不好是因为他不努力。
建议：对他加强考核。
```
*问题：自利偏差+基本归因错误导致双重标准*

✅ **After（正确做法）**
```markdown
## 分析结论
### 归因一致性检查
| 情况 | 归因 | 是否一致 |
|------|------|---------|
| 自己业绩不好 | 市场差(情境) | - |
| 他人业绩不好 | 不努力(个人) | 不一致 |

### 建议
1. 使用相同归因标准评估自己和他人
2. 如市场差影响自己，也影响他人
```
*优势：归因一致性检查，识别双重标准*

**原因**：基本归因错误常与自利偏差叠加。自己失败归情境（自利），他人失败归个人（基本归因）。

## 陷阱 3：忽视行为-情境交互

**问题**：将个人和情境视为独立因素，忽视交互作用。

❌ **Before（错误做法）**
```markdown
## 分析结论
个人因素占30%，情境因素占70%，综合评估。
建议：按比例归因。
```
*问题：将个人和情境简单加权，忽视交互作用*

✅ **After（正确做法）**
```markdown
## 分析结论
### 交互分析
1. 市场环境差(情境) x 应对能力弱(个人) = 业绩大幅下滑
2. 市场环境差(情境) x 应对能力强(个人) = 业绩小幅下滑
3. 关键杠杆点：提升应对能力可显著改善结果

### 建议
1. 识别高杠杆的个人能力（竞品分析、客户沟通）
2. 针对性培训提升这些能力
```
*优势：识别个人-情境交互，找到杠杆点*

**原因**：行为是个人和情境交互的结果，非简单加权。交互分析揭示杠杆点。
