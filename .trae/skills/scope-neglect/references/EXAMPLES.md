# 分析示例

## 示例 1：范围忽视——环保捐赠决策

**输入**：这个湖泊有2000只鸟因石油泄漏死亡，我们应该捐款救助，预计需要10万元。

```
Step 1: 识别范围信号 → 产出：发现"2000只鸟...10万元"表述，情感响应与规模不成比例
Step 2: 评估范围敏感性 → 产出：2000只鸟和20000只鸟触发的捐赠意愿可能相似，范围不敏感
Step 3: 分析范围机制 → 产出：情感启发：个体故事触发共情，但规模扩大不线性增加情感；心理账户：捐赠预算有上限
Step 4: 校正范围忽视 → 产出：建议用成本效益评估，而非情感响应
```

**输出**：
```
## 分析结论
### 核心发现
1. 2000只鸟触发捐赠意愿，但20000只鸟可能触发相似意愿（范围不敏感）
2. Desvousges等(1993)实验：人们愿为2000只鸟和200000只鸟支付相似金额
3. 情感响应不随规模线性增长

### 建议（按优先级）
1. [高优先级] 用成本效益评估：每元拯救的鸟数
2. [高优先级] 理解情感响应的范围不敏感性
3. [中优先级] 建立捐赠评估框架

### 关键假设
- 成本效益可量化
- 情感响应范围不敏感

### 风险
- 范围忽视可能导致资源分配不均
- 需要理性框架辅助情感决策
```

## 示例 2：范围忽视——安全投入决策

**输入**：这个安全漏洞会影响100个用户，我们需要投入50万修复。

```
Step 1: 识别范围信号 → 产出：发现"100个用户...50万"表述，需评估投入与影响范围是否匹配
Step 2: 评估范围敏感性 → 产出：100用户和10000用户可能触发相似投入意愿，范围不敏感
Step 3: 分析范围机制 → 产出：情感响应：安全问题触发恐惧，但规模扩大不线性增加恐惧；固定预算：安全预算有上限
Step 4: 校正范围忽视 → 产出：建议用风险量化评估投入
```

**输出**：
```
## 分析结论
### 核心发现
1. 100用户影响，50万修复成本，每用户成本5000元
2. 范围忽视使100用户和10000用户可能触发相似投入
3. 需评估风险严重程度和修复成本效益

### 建议（按优先级）
1. [高优先级] 量化风险：影响范围×严重程度×概率
2. [高优先级] 评估修复成本效益：50万 vs 潜在损失
3. [中优先级] 建立安全投入优先级框架

### 关键假设
- 风险可量化
- 成本效益可评估

### 风险
- 范围忽视可能导致过度投入或投入不足
- 需要数据驱动决策
```

## 反例：不该触发此 Skill 的场景

**输入**：这个漏洞影响100万用户，每用户损失100元，总风险1亿元，投入500万修复很划算。

**原因**：这是基于量化风险评估（1亿风险vs500万修复），不是范围忽视。范围忽视指情感响应与规模不成比例。
