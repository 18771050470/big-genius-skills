# 常见陷阱

## 陷阱 1：情感响应与规模不成比例

**问题**：对大小规模问题的情感响应相似。

❌ **Before（错误做法）**
```markdown
## 分析结论
2000只鸟死亡，我们应该捐10万。
建议：立即捐赠。
```
*问题：范围忽视使2000只和200000只触发相似捐赠意愿*

✅ **After（正确做法）**
```markdown
## 分析结论
### 范围敏感性分析
| 规模 | 情感响应 | 理性评估 |
|------|---------|---------|
| 2000只 | 强 | 每元拯救鸟数 |
| 200000只 | 强(相似) | 每元拯救鸟数(更高) |

### 建议
1. 情感响应不随规模增长
2. 用成本效益评估捐赠效率
```
*优势：量化评估替代情感响应*

**原因**：Desvousges等(1993)：人们愿为2000、20000、200000只鸟支付相似金额（约80美元）。情感响应范围不敏感。

## 陷阱 2：固定预算启发

**问题**：为某类问题设置固定预算，不随规模调整。

❌ **Before（错误做法）**
```markdown
## 分析结论
环保项目预算100万，不管规模大小。
建议：按预算执行。
```
*问题：范围忽视使预算不随问题规模调整*

✅ **After（正确做法）**
```markdown
## 分析结论
### 规模vs预算
| 规模 | 影响 | 建议预算 |
|------|------|---------|
| 小(1000只) | 低 | 10万 |
| 中(10000只) | 中 | 50万 |
| 大(100000只) | 高 | 200万 |

### 建议
1. 预算应随问题规模调整
2. 用成本效益评估
```
*优势：规模敏感预算*

**原因**：人们为环保、慈善等设置"心理预算"，不随问题规模变化。

## 陷阱 3：个体故事替代规模数据

**问题**：被个体故事打动，忽视规模数据。

❌ **Before（错误做法）**
```markdown
## 分析结论
这只鸟的照片太可怜了，我们要救助。
建议：优先救助。
```
*问题：范围忽视使个体故事替代规模评估*

✅ **After（正确做法）**
```markdown
## 分析结论
### 个体vs规模
| 维度 | 评估 |
|------|------|
| 个体故事 | 强情感响应 |
| 规模数据 | 2000只需要系统方案 |
| 成本效益 | 每元拯救鸟数 |

### 建议
1. 个体故事触发情感，但规模需要系统方案
2. 用数据指导资源分配
```
*优势：系统方案替代个体响应*

**原因**：可识别受害者效应+范围忽视：个体故事触发强情感，但规模问题需要理性评估。
