## 交付物模板 3：云安全配置检查清单

```markdown
# 云安全配置检查清单（AWS/Azure/GCP）

## 身份与访问管理
- [ ] 根账号启用 MFA
- [ ] 根账号不使用日常操作
- [ ] IAM 用户/角色遵循最小权限
- [ ] 定期轮换访问密钥（90 天）
- [ ] 启用 CloudTrail/Activity Log
- [ ] 使用 IAM Role 而非长期凭证

## 存储
- [ ] S3/Storage Account 禁止公开访问
- [ ] 启用服务端加密（SSE-S3/KMS）
- [ ] 启用版本控制
- [ ] 配置生命周期策略
- [ ] 启用访问日志

## 网络
- [ ] 安全组默认拒绝
- [ ] 数据库不暴露公网
- [ ] 使用 VPC/Virtual Network 分段
- [ ] 启用 Flow Log
- [ ] 使用 WAF/Front Door

## 计算
- [ ] 实例不使用长期凭证
- [ ] 启用自动补丁更新
- [ ] 配置安全组最小开放
- [ ] 启用实例元数据保护

## 监控
- [ ] 启用 GuardDuty/Security Center
- [ ] 配置告警规则
- [ ] 日志保留 >= 1 年
- [ ] 定期审计配置漂移
```
