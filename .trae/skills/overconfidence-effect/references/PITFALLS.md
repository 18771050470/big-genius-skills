# 常见陷阱

## 陷阱 1：优于平均效应

**问题**：认为自己在各方面都优于平均水平。

❌ **Before（错误做法）**
```markdown
## 分析结论
我学新技术比一般人快，2周就能掌握。
建议：按2周计划。
```
*问题：过度自信使优于平均效应导致低估时间*

✅ **After（正确做法）**
```markdown
## 分析结论
### 自信vs数据对比
| 维度 | 自我评估 | 实际数据 |
|------|---------|---------|
| 学习速度 | 快于80%的人 | 平均水平 |
| 掌握时间 | 2周 | 6周(历史) |

### 建议
1. 80-90%人认为优于平均，数学上不可能
2. 基于历史数据：6周
```
*优势：数据校准优于平均效应*

**原因**：Svenson(1981)实验：93%美国司机认为自己驾驶技术优于平均。优于平均效应在各领域普遍存在。

## 陷阱 2：控制错觉

**问题**：高估自己对结果的控制力。

❌ **Before（错误做法）**
```markdown
## 分析结论
我们创业成功率80%，团队经验丰富。
建议：全力投入。
```
*问题：过度自信使高估控制力*

✅ **After（正确做法）**
```markdown
## 分析结论
### 可控vs不可控因素
| 因素 | 可控性 | 影响 |
|------|--------|------|
| 团队经验 | 高 | 中 |
| 产品质量 | 高 | 高 |
| 市场环境 | 低 | 高 |
| 竞争对手 | 低 | 高 |
| 运气 | 无 | 高 |

### 建议
1. 不可控因素占50%+影响
2. 基于基准率：50%成功率
```
*优势：区分可控和不可控因素*

**原因**：Langer(1975)控制错觉：人们高估自己对不可控事件的控制力，尤其在有选择权和熟悉度时。

## 陷阱 3：确认偏误强化自信

**问题**：只回忆成功案例，忽视失败案例。

❌ **Before（错误做法）**
```markdown
## 分析结论
之前学类似技术都很快，这次也一样。
建议：乐观估计。
```
*问题：确认偏误使只回忆成功案例*

✅ **After（正确做法）**
```markdown
## 分析结论
### 完整历史回顾
| 案例 | 结果 | 时间 |
|------|------|------|
| 技术A | 成功 | 4周 |
| 技术B | 成功 | 3周 |
| 技术C | 困难 | 8周 |
| 技术D | 失败 | 放弃 |

### 建议
1. 平均时间6周，非"都很快"
2. 有失败案例，需警惕
```
*优势：完整历史替代选择性回忆*

**原因**：确认偏误使人们选择性回忆支持自信的案例，忽视反面证据。
