# 常见陷阱

## 陷阱 1：将不幸归因于受害者自身

**问题**：因不幸发生而假设受害者一定有问题。

❌ **Before（错误做法）**
```markdown
## 分析结论
小王被裁员，肯定是他工作不努力。
建议：我们要更努力工作，避免被裁。
```
*问题：公正世界假设使将不幸归因于受害者，维持"世界公正"信念*

✅ **After（正确做法）**
```markdown
## 分析结论
### 多因素归因
| 因素 | 影响程度 | 证据 |
|------|---------|------|
| 公司战略调整 | 高 | 整个部门裁撤30% |
| 部门预算削减 | 中 | 预算减少40% |
| 个人绩效 | 低 | 过去4季度B+，中等偏上 |

### 建议
1. 裁员主因是系统性决策，非个人失败
2. 理解不幸不一定是受害者过错
```
*优势：多因素归因揭示系统性原因，避免责备受害者*

**原因**：Lerner(1980)证明：人们倾向于责备受害者以维持"世界公正"信念。这提供虚假安全感——只要不犯同样错就不会受害。

## 陷阱 2：将成功归因于个人能力

**问题**：因成功发生而假设成功者一定有能力。

❌ **Before（错误做法）**
```markdown
## 分析结论
他创业成功，肯定是能力超强。
建议：学习他的成功经验。
```
*问题：公正世界假设使将成功归因于能力，忽视运气和时机*

✅ **After（正确做法）**
```markdown
## 分析结论
### 成功因素分析
| 因素 | 影响程度 | 说明 |
|------|---------|------|
| 市场时机 | 高 | 进入时市场快速增长 |
| 融资环境 | 中 | 当时资本充裕 |
| 个人能力 | 中 | 执行力强，但非唯一因素 |
| 运气 | 中 | 竞品同时期出问题 |

### 建议
1. 成功是多因素结果，能力只是其一
2. 学习可复制的因素（执行力），接受不可控因素（运气）
```
*优势：多因素分析揭示成功复杂性，避免能力归因偏差*

**原因**：公正世界假设使成功被归因于能力，失败被归因于缺陷。但运气和时机起重要作用。

## 陷阱 3：忽视随机因素

**问题**：将所有结果归因于可控因素，忽视随机性。

❌ **Before（错误做法）**
```markdown
## 分析结论
只要努力就能成功，失败就是因为不够努力。
建议：加倍努力。
```
*问题：公正世界假设使忽视随机因素，将一切归因于努力*

✅ **After（正确做法）**
```markdown
## 分析结论
### 结果=努力+能力+运气
| 因素 | 可控性 | 影响程度 |
|------|--------|---------|
| 努力 | 高 | 中 |
| 能力 | 中 | 高 |
| 运气 | 低 | 高 |

### 建议
1. 努力和能力可控，应聚焦
2. 运气不可控，需管理风险
3. 接受随机性，不因失败自责过度
```
*优势：承认随机因素，平衡可控和不可控因素*

**原因**：公正世界假设使世界看起来完全公正，但随机性在结果中起重要作用。
