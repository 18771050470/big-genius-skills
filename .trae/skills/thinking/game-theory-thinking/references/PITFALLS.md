# 常见陷阱

## 陷阱 1：假设所有参与者完全理性

**反例**：
```
"竞品不会降价，因为降价对他们不利"（但竞品可能非理性或信息不对称）
```

**正例**：
```
"假设竞品理性，但考虑20%概率非理性行为，调整策略"
```

**原因**：现实中的参与者常有行为偏差。

---

## 陷阱 2：忽略重复博弈效应

**反例**：
```
用单次博弈分析长期竞争关系
```

**正例**：
```
"这是重复博弈，需要考虑声誉和触发策略"
```

**原因**：重复博弈中合作可能成为均衡。

---

## 陷阱 3：收益矩阵未量化

**反例**：
```
"降价对我们不好"（多不好？量化缺失）
```

**正例**：
```
"降价20%导致利润-30%，份额+5%"
```

**原因**：无法量化就无法比较。
