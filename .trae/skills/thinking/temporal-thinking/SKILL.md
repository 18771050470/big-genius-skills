# 时间思维 (Temporal Thinking)

## 核心定义

时间思维是一种结构化的认知框架，要求从时间维度拆解问题：通过回顾过去（历史因果）理解现状的成因，通过分析现状（当前状态）识别未来的趋势，通过预判未来（趋势推演）指导当前的行动，并时刻警惕行动与结果之间的延迟效应。

## 思考流程控制

### 中断处理

- **信息不足**：用 AskUserQuestion 向用户提问补充
- **发现矛盾**：停止思考，用 AskUserQuestion 向用户报告矛盾点，等待决策
- **提前终止**：若某步已得出明确结论且后续步骤无增量价值，可提前进入总结输出，但须标注"提前终止，跳过 Step N+1"

## 思考流程

### Step 1: 情境时间坐标定位

- 明确问题的核心变量，并确定其观察的时间尺度（短、中、长期）
- 定义“过去”的起点（如项目启动时间、系统上线时间）、终点（当前）
- 定义“未来”的预测窗口（如1年、5年、10年）
- 一个带有明确时间坐标的问题定义

### Step 2: 回溯因果链：从过去到现在

- 逆向追溯。列出主要的事件、决策和“关键时刻”
- 识别哪些是【核心因】（直接导致现状）、【辅助因】（创造条件）、【触发因】（引爆点）
- 识别【延迟效应】：某决策在实施多久后产生了效果？存在哪些未预料到的延迟结果？
- 一个包含原因-结果关系和时间延迟标注的历史事件链

### Step 3: 解析当前状态：存量与流量

- 分析“现在”这个瞬时横截面
- 识别【存量】(Stock)：当前积累的资源、能力、债务、技术债、人脉等
- 识别【流量】(Flow)：流入/流出这些存量的速率，如用户增长率、债务偿还速度、员工离职率
- 理解哪些存量的变化是由哪些流量在过去一段时间内累积而成的
- 当前系统状态的“资产负债表”，清晰列出存量和关键流量的当前数值及变化方向

### Step 4: 前瞻推演：从现在到未来

- 基于当前存量和流量，对未来状态进行推演
- 路径 A：按现有流量和结构，推演“默认未来” (The Default Future)
- 路径 B：假设改变某个关键流量或结构，推演“干预未来” (The Intervened Future)
- 应用【延迟效应】分析：一个干预动作多久后才会在系统中体现出来？短期效果和长期效果是否可能矛盾？
- 2-3种可能的未来情景及达成该情景所需的关键条件，同时标注出每种情景的“时间延迟”特征

### Step 5: 进行决策与行动规划

- 结合未来预判，评估当前应该做什么，不应该做什么
- 识别具备“时间杠杆效应”的干预点：在哪个时间点、改变哪个流量，能以最小的成本产生最大的长期正向效应？
- 制定分阶段行动计划，并设置“时间检查点”用来验证趋势是否符合预期
- 一份包含短期行动、长期目标，并内置了反馈验证机制的行动计划

## 关键原则

1.  **从过去出发，回到过去（因果原则）**：不要孤立地看待问题，要把它看作是历史长河中的一个点。**原因**：不理解过去，就无法解释现在，更无法预测未来。所有现状都是历史决策和外部事件累积的结果。
2.  **立足现在，放眼未来（趋势原则）**：当前的状态不仅是结果，也是未来的起点。**原因**：决策的价值在于改变未来，所以必须基于对当前“存量”和“流量”的真实评估，来推演未来可能的演化路径。
3.  **理解延迟效应（时差原则）**：行动和结果之间永远存在时间差。**原因**：这是许多决策失败的根本原因。忽略了延迟，就会在短期未见成效时错误地放弃正确的策略，或是在坏结果显现前持续加码错误的策略。
4.  **不求精准预测，但求推演可能（情景原则）**：未来不可预测，但可以准备。**原因**：时间思维的目标不是提供一个关于未来的唯一答案，而是构建几种合理的未来情景，帮助你提前识别风险、准备预案，从而在今天做出更具韧性的决策。

## 自检清单

> 聚焦时间思维最容易被误用的环节

**执行规则**：
- **执行方**：Agent 自检，在最终输出前逐项确认
- **未通过处理**：任一检查项未通过，必须修正后重新自检，禁止跳过
- **强制停止项**：以下检查项未通过时，必须停止输出并修正：
  - [ ] 已按思考流程完成全部步骤，无跳过（或已标注提前终止原因）
  - [ ] 输出具体可执行，不含模糊建议
  - [ ] 标注了关键假设和不确定性

**时间思维特有关键检查项**：

- [ ] **时间范围是否清晰界定？** 过去的起点、未来的终点、预测窗口是否被明确定义？
- [ ] **是否列出了至少3个关键历史事件？** 因果链是否被充分追溯？
- [ ] **是否识别了至少1个显著的延迟效应？** 行动与结果之间的时间差是否被量化？
- [ ] **原因类型是否被区分？** 核心因、辅助因、触发因是否被混为一谈？
- [ ] **存量与流量是否被清晰描述？** 当前积累的资源/债务和变化速率是否被准确刻画？
- [ ] **是否推演了至少2种未来情景？** 默认未来和干预未来是否都被考虑？
- [ ] **干预的时间杠杆效应是否被识别？** 在哪个时间点改变哪个流量能产生最大长期效应？
- [ ] **行动计划是否包含验证点？** 趋势是否如期发生的检查机制是否被内置？
