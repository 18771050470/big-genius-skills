# 常见陷阱

## 陷阱 1：用后见之明评估历史决策

**反例**：
```
"当时就应该知道这个技术不成熟"（但当时没有这个信息）
```

**正例**：
```
"决策时该技术已有2个成功案例，但也有3个失败案例，信息是混合的"
```

**原因**：只能用决策时已知信息评估。

---

## 陷阱 2：构建不可能的反事实

**反例**：
```
"如果当时有10倍预算就好了"（不现实）
```

**正例**：
```
"如果当时增加20%预算用于技术预研"（可行）
```

**原因**：反事实必须在当时是可行的。

---

## 陷阱 3：混淆运气和决策质量

**反例**：
```
"结果好=决策好"（可能只是运气好）
```

**正例**：
```
"结果好，但决策过程有缺陷，下次可能没这么好运气"
```

**原因**：短期结果受运气影响大，长期期望值才能评估决策质量。

---

## 陷阱 4：沉溺于后悔而非学习

**反例**：
```
"早知道就..."（情绪化，无建设性）
```

**正例**：
```
"下次遇到类似情况，我会增加X检查点"（可操作）
```

**原因**：反事实的目的是改进未来决策，不是后悔过去。
