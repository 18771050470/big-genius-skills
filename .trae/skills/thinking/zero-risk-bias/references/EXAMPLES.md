# 分析示例

## 示例 1：零风险偏差——食品安全决策

**输入**：这个食品添加剂有0.01%的致癌风险，必须完全禁用，不能有任何风险。

```
Step 1: 识别零风险信号 → 产出：发现"必须完全禁用...不能有任何风险"表述，追求零风险而非风险降低
Step 2: 评估风险降低 → 产出：禁用成本：食品保质期缩短50%，浪费增加30%；风险降低：0.01%→0%
Step 3: 分析零风险机制 → 产出：确定性偏好：零风险被赋予不成比例的价值；风险厌恶：任何风险都被视为不可接受
Step 4: 校正零风险偏差 → 产出：建议比较风险降低的成本效益，而非追求零风险
```

**输出**：
```
## 分析结论
### 核心发现
1. 0.01%致癌风险极低，禁用成本远高于风险降低收益
2. 零风险偏差使"零"被赋予不成比例的价值
3. Viscusi等(1987)：人们愿为从1%降到0%支付比从5%降到1%更多

### 建议（按优先级）
1. [高优先级] 比较成本效益：禁用成本 vs 风险降低收益
2. [高优先级] 理解零风险偏差：从1%到0%的心理价值被放大
3. [中优先级] 设置可接受风险阈值，而非追求零风险

### 关键假设
- 风险数据可靠
- 成本可量化

### 风险
- 零风险偏差可能导致资源浪费
- 需要平衡风险和收益
```

## 示例 2：零风险偏差——软件安全

**输入**：这个系统必须100%安全，不能有任何漏洞，否则后果不堪设想。

```
Step 1: 识别零风险信号 → 产出：发现"必须100%安全...不能有任何漏洞"表述，追求绝对安全
Step 2: 评估安全投入 → 产出：100%安全成本：无限投入；99%安全成本：合理投入，覆盖99%风险
Step 3: 分析零风险机制 → 产出：确定性偏好：100%安全被过度估值；恐惧驱动：安全事件恐惧使追求绝对安全
Step 4: 校正零风险偏差 → 产出：建议设置合理安全目标，平衡安全和成本
```

**输出**：
```
## 分析结论
### 核心发现
1. 100%安全在技术上不可能，成本无限
2. 99%安全可覆盖绝大多数风险，成本合理
3. 零风险偏差使追求不可能目标，浪费资源

### 建议（按优先级）
1. [高优先级] 设置合理安全目标：99.9%可用性，关键漏洞0
2. [高优先级] 基于风险评估分配安全预算
3. [中优先级] 接受残余风险，制定应急响应

### 关键假设
- 100%安全不可能
- 残余风险可管理

### 风险
- 零风险偏差可能导致安全投入过度
- 需要平衡安全和业务需求
```

## 反例：不该触发此 Skill 的场景

**输入**：这个核反应堆安全系统必须达到99.9999%可靠性，因为失败后果是灾难性的。

**原因**：这是基于后果严重性的合理高安全要求（核安全），不是零风险偏差。零风险偏差指追求零风险而非风险优化。
