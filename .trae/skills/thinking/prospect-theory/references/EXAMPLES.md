# 分析示例

## 示例 1：前景理论——保险购买决策

**输入**：这个保险年费5000元，生病概率只有1%，但万一得了大病就倾家荡产，还是买吧。

```
Step 1: 识别前景信号 → 产出：发现"万一...倾家荡产"表述，小概率大损失被过度加权
Step 2: 评估概率权重 → 产出：1%概率被主观加权为5-10%，损失厌恶使损失权重是收益的2.5倍
Step 3: 分析前景机制 → 产出：Kahneman & Tversky(1979)：小概率过度加权+损失厌恶+参照点依赖
Step 4: 校正前景偏差 → 产出：建议用期望值评估，识别概率加权偏差
```

**输出**：
```
## 分析结论
### 核心发现
1. 1%概率被过度加权，损失厌恶使保险购买决策偏向保守
2. 期望值计算：5000元保费 vs 1%×50万=5000元期望损失，基本等价
3. 但心理账户使"保费确定损失"比"可能的大病损失"更被规避

### 建议（按优先级）
1. [高优先级] 期望值基本等价，购买决策取决于风险偏好
2. [高优先级] 理解小概率被过度加权，1%≠心理感知的1%
3. [中优先级] 评估保险条款实际覆盖范围

### 关键假设
- 大病治疗费用估计准确
- 概率数据可靠

### 风险
- 前景理论使小概率大损失被过度关注
- 需要区分理性保险和过度保险
```

## 示例 2：前景理论——产品定价策略

**输入**：这个功能降价100元用户应该很高兴，比涨价100元用户不高兴的影响小。

```
Step 1: 识别前景信号 → 产出：发现"降价高兴<涨价不高兴"表述，损失厌恶使价格变化不对称
Step 2: 评估损失权重 → 产出：涨价100元(损失)痛苦是降价100元(收益)快乐的2.5倍
Step 3: 分析前景机制 → 产出：参照点依赖：当前价格为参照；损失厌恶：损失>收益；敏感度递减：绝对值影响减弱
Step 4: 校正前景偏差 → 产出：建议理解价格变化不对称影响
```

**输出**：
```
## 分析结论
### 核心发现
1. 涨价100元的不满意度是降价100元满意度的2.5倍
2. 参照点依赖：用户以当前价格为参照，涨价被视为损失
3. 价格策略应考虑损失厌恶：小幅渐进涨价优于一次性大幅涨价

### 建议（按优先级）
1. [高优先级] 涨价影响远大于降价，需谨慎
2. [高优先级] 用增值服务包装涨价，改变参照点
3. [中优先级] 分阶段涨价，降低损失感知

### 关键假设
- 损失厌恶系数2.5适用于价格场景
- 用户参照点是当前价格

### 风险
- 损失厌恶使涨价导致用户流失
- 需要平衡收入和用户满意度
```

## 反例：不该触发此 Skill 的场景

**输入**：这个产品定价基于成本加成20%，符合行业标准。

**原因**：这是成本定价法，不涉及用户心理感知。前景理论适用于涉及风险、损失、参照点的决策场景。
