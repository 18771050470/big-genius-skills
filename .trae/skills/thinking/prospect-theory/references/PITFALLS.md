# 常见陷阱

## 陷阱 1：小概率过度加权

**问题**：将小概率事件当作大概率来对待。

❌ **Before（错误做法）**
```markdown
## 分析结论
万一大病就倾家荡产，必须买保险。
建议：买最贵的保险。
```
*问题：前景理论使1%概率被过度加权*

✅ **After（正确做法）**
```markdown
## 分析结论
### 概率加权分析
| 维度 | 值 |
|------|-----|
| 客观概率 | 1% |
| 主观加权 | 5-10% |
| 期望损失 | 5000元 |
| 保费 | 5000元 |

### 建议
1. 期望值等价，购买取决于风险偏好
2. 理解主观概率被放大
```
*优势：区分客观概率和主观加权*

**原因**：Tversky & Kahneman(1992)：1%概率被主观加权为约5-10%，小概率事件被过度重视。

## 陷阱 2：损失厌恶

**问题**：损失痛苦远大于收益快乐。

❌ **Before（错误做法）**
```markdown
## 分析结论
涨价100元用户会不高兴，但降价100元用户会高兴，应该多降价。
建议：频繁降价。
```
*问题：未理解损失厌恶使涨价影响更大*

✅ **After（正确做法）**
```markdown
## 分析结论
### 损失vs收益权重
| 变化 | 心理影响 | 权重 |
|------|---------|------|
| 涨价100元 | 损失痛苦 | 2.5 |
| 降价100元 | 收益快乐 | 1.0 |

### 建议
1. 涨价负面影响是降价正面影响的2.5倍
2. 涨价需谨慎，可用增值服务包装
```
*优势：量化损失厌恶影响*

**原因**：Kahneman & Tversky(1979)：损失厌恶系数约2.5，损失100元的痛苦是获得100元快乐的2.5倍。

## 陷阱 3：参照点依赖

**问题**：忽视参照点对决策的影响。

❌ **Before（错误做法）**
```markdown
## 分析结论
这个产品从1000元降到800元，用户应该觉得便宜。
建议：降价促销。
```
*问题：未考虑用户参照点可能是竞品价格而非原价*

✅ **After（正确做法）**
```markdown
## 分析结论
### 参照点分析
| 参照点 | 感知 |
|--------|------|
| 原价1000元 | 便宜200元(收益) |
| 竞品600元 | 贵200元(损失) |
| 用户实际参照 | 可能是竞品 |

### 建议
1. 用户参照点可能不是原价
2. 需了解用户真实参照点
```
*优势：识别真实参照点*

**原因**：参照点决定收益/损失感知。不同参照点导致完全不同的决策。
