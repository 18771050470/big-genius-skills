# 分析示例

## 示例 1：信息偏差——过度调研决策

**输入**：这个产品方向我们还需要更多数据，再调研2周，收集更多用户反馈再决定。

```
Step 1: 识别信息信号 → 产出：发现"需要更多数据"、"再调研2周"表述，信息收集成为决策延迟理由
Step 2: 评估信息价值 → 产出：已有数据：200份问卷、50次访谈；新增数据边际价值递减
Step 3: 分析信息机制 → 产出：信息错觉：更多信息=更好决策；确定性追求：用信息收集缓解不确定焦虑
Step 4: 校正信息偏差 → 产出：建议评估信息边际价值，设定决策截止时间
```

**输出**：
```
## 分析结论
### 核心发现
1. 已有数据量充足（200份问卷+50次访谈），新增信息边际价值低
2. 2周调研成本：人力成本3万+机会成本（延迟上线损失10万）
3. 核心问题已清晰：用户需求明确，不确定性在执行层面非信息层面

### 建议（按优先级）
1. [高优先级] 基于现有数据决策，设定本周内确定方向
2. [高优先级] 用MVP验证剩余不确定性，而非继续调研
3. [中优先级] 建立"足够信息"标准，避免无限调研

### 关键假设
- 现有数据样本具有代表性
- MVP可验证剩余不确定性

### 风险
- 继续调研导致机会成本增加
- 延迟上线可能错过市场窗口
```

## 示例 2：信息偏差——医疗诊断过度检查

**输入**：这个症状还需要做更多检查，CT、MRI、血液全做一遍，确保万无一失。

```
Step 1: 识别信息信号 → 产出：发现"全做一遍"表述，信息收集超过诊断需要
Step 2: 评估信息价值 → 产出：初步检查已指向明确诊断方向，额外检查边际价值低
Step 3: 分析信息机制 → 产出：防御性医疗：用检查规避责任；确定性追求：用更多信息缓解焦虑
Step 4: 校正信息偏差 → 产出：建议基于诊断概率选择检查，而非全面筛查
```

**输出**：
```
## 分析结论
### 核心发现
1. 初步检查已指向A诊断(概率80%)，额外检查对诊断改变概率<5%
2. 全面检查成本：费用5000元+时间2天+假阳性风险15%
3. 假阳性可能导致不必要的后续检查和治疗

### 建议（按优先级）
1. [高优先级] 基于诊断概率选择检查：先做最可能的A诊断确认检查
2. [高优先级] 评估假阳性风险：15%假阳性率可能导致过度治疗
3. [中优先级] 与患者沟通风险收益，共同决策

### 关键假设
- 初步检查可靠性高
- 假阳性风险可量化

### 风险
- 过度检查增加成本和假阳性风险
- 假阳性可能导致不必要的焦虑和治疗
```

## 反例：不该触发此 Skill 的场景

**输入**：这个关键决策我们只有30%的信息，需要收集更多数据再决定。

**原因**：这是信息不足时的合理数据收集，不是信息偏差。信息偏差指已有足够信息仍继续收集。
