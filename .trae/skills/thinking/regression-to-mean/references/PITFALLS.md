# 常见陷阱

## 陷阱 1：期望极端表现持续

**问题**：因一次极端好表现而期望持续。

❌ **Before（错误做法）**
```markdown
## 分析结论
上月业绩第一，这个月肯定还是优秀。
建议：给最高目标。
```
*问题：忽视回归均值，期望极端表现持续*

✅ **After（正确做法）**
```markdown
## 分析结论
### 回归数据分析
| 指标 | 值 |
|------|-----|
| 上月排名 | 第1 |
| 历史下月平均排名 | 第8 |
| 回归幅度 | 7位 |

### 建议
1. 极端表现后趋向平均是统计规律
2. 基于长期平均设置目标
```
*优势：回归数据校准期望*

**原因**：Galton(1886)发现：高个子父母的孩子身高趋向平均。极端表现后回归平均是普遍统计现象。

## 陷阱 2：将回归误解为能力下降

**问题**：极端表现后回归被误认为能力问题。

❌ **Before（错误做法）**
```markdown
## 分析结论
上月第一，这月掉到第10，是不是态度有问题？
建议：加强考核。
```
*问题：将统计回归误解为能力下降*

✅ **After（正确做法）**
```markdown
## 分析结论
### 回归vs能力下降
| 指标 | 评估 |
|------|------|
| 过去6月平均 | 第5名 |
| 本月排名 | 第10名 |
| 回归预期 | 正常(从第1回归) |

### 建议
1. 从第1回归到第10是正常统计现象
2. 长期平均第5名反映真实水平
```
*优势：区分统计回归和能力变化*

**原因**：Kahneman(2011)：回归均值常被误解为因果解释（如"骄傲导致下滑"），实际是统计必然。

## 陷阱 3：因果归因错误

**问题**：为回归现象寻找因果解释。

❌ **Before（错误做法）**
```markdown
## 分析结论
基金去年涨80%，今年跌了，肯定是基金经理换了策略。
建议：卖出。
```
*问题：将回归均值归因于具体原因*

✅ **After（正确做法）**
```markdown
## 分析结论
### 回归vs因果
| 解释 | 可能性 |
|------|--------|
| 回归均值 | 高(统计规律) |
| 策略变化 | 低(无证据) |
| 经理变更 | 低(无证据) |

### 建议
1. 极端收益后回归是统计规律
2. 无需寻找具体因果解释
```
*优势：识别统计规律，避免过度归因*

**原因**：人们倾向于为统计现象寻找因果解释，但回归均值无需特定原因。
