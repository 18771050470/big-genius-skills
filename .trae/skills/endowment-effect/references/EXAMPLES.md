# 分析示例

## 示例 1：禀赋效应——二手设备估值

**输入**：这台电脑我用了2年，当初8000买的，现在至少卖5000，毕竟我用得很爱惜。

```
Step 1: 识别禀赋信号 → 产出：发现"至少卖5000"、"用得很爱惜"表述，拥有者高估物品价值
Step 2: 评估估值偏差 → 产出：市场二手价约3000，拥有者估值5000，溢价67%
Step 3: 分析禀赋机制 → 产出：损失厌恶：失去物品的痛苦>获得的快乐；现状偏好：拥有的物品被赋予额外价值
Step 4: 校正禀赋效应 → 产出：建议参考市场价格，而非个人情感估值
```

**输出**：
```
## 分析结论
### 核心发现
1. 拥有者估值5000，市场二手价约3000，溢价67%
2. Kahneman等(1990)咖啡杯实验：拥有者要价是中位数的2倍
3. "用得很爱惜"是情感附加，市场不为此付费

### 建议（按优先级）
1. [高优先级] 参考同类二手产品市场价：3000-3500
2. [高优先级] 区分情感价值和市场价值
3. [中优先级] 设置合理预期，避免定价过高导致滞销

### 关键假设
- 二手市场数据可靠
- 产品功能正常

### 风险
- 定价过高导致长期滞销
- 需要区分情感价值和市场价值
```

## 示例 2：禀赋效应——股权保留决策

**输入**：我们公司的股票不能卖，这是我们的，即使现在有更好的投资机会也要持有。

```
Step 1: 识别禀赋信号 → 产出：发现"不能卖"、"这是我们的"表述，因拥有而高估当前资产
Step 2: 评估估值偏差 → 产出：当前股票预期收益8%，新投资机会预期收益15%，但禀赋效应使不愿换仓
Step 3: 分析禀赋机制 → 产出：现状偏好：当前持有被赋予额外价值；损失厌恶：卖出被视为"失去"
Step 4: 校正禀赋效应 → 产出：建议比较预期收益，而非基于拥有状态决策
```

**输出**：
```
## 分析结论
### 核心发现
1. 当前股票预期收益8%，新投资机会15%，差距7%
2. 禀赋效应使"拥有的"被高估，不愿换仓
3. 10年后复利差距：(1.08)^10=2.16 vs (1.15)^10=4.05，差距近2倍

### 建议（按优先级）
1. [高优先级] 基于预期收益决策，而非拥有状态
2. [高优先级] 考虑换仓的税务成本后，净收益仍更高
3. [中优先级] 分批换仓，降低心理阻力

### 关键假设
- 预期收益估计可靠
- 税务成本可计算

### 风险
- 禀赋效应可能导致机会成本损失
- 需要克服"失去"的心理障碍
```

## 反例：不该触发此 Skill 的场景

**输入**：这台电脑是限量版，市场稀缺，所以估值高于普通二手价。

**原因**：这是基于稀缺性的合理溢价，不是禀赋效应。禀赋效应指仅因"拥有"而高估。
