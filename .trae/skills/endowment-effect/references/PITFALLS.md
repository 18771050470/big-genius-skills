# 常见陷阱

## 陷阱 1：因拥有而高估物品价值

**问题**：将拥有状态等同于更高价值。

❌ **Before（错误做法）**
```markdown
## 分析结论
这台电脑我用的很爱惜，至少卖5000。
建议：定价5000。
```
*问题：禀赋效应使拥有者高估价值，市场不认可*

✅ **After（正确做法）**
```markdown
## 分析结论
### 估值对比
| 估值来源 | 价格 | 说明 |
|---------|------|------|
| 拥有者估值 | 5000 | 含情感附加 |
| 市场二手价 | 3000 | 客观市场价 |
| 差距 | 67% | 禀赋效应溢价 |

### 建议
1. 市场不为情感价值付费
2. 合理定价3000-3500
```
*优势：区分情感价值和市场价值*

**原因**：Kahneman等(1990)实验：拥有咖啡杯的人要价中位数7.12美元，非拥有者只愿付2.87美元。拥有使价值感知翻倍。

## 陷阱 2：现状偏好

**问题**：因当前持有而不愿改变，即使有更好的选择。

❌ **Before（错误做法）**
```markdown
## 分析结论
我们的股票不能卖，即使有更好的投资机会。
建议：继续持有。
```
*问题：禀赋效应使现状偏好替代收益比较*

✅ **After（正确做法）**
```markdown
## 分析结论
### 收益比较
| 选项 | 预期收益 | 10年复利 |
|------|---------|---------|
| 当前股票 | 8% | 2.16倍 |
| 新投资机会 | 15% | 4.05倍 |
| 差距 | 7% | 近2倍 |

### 建议
1. 基于预期收益决策，而非拥有状态
2. 禀赋效应使不愿换仓，但机会成本真实存在
```
*优势：用收益比较替代现状偏好*

**原因**：Samuelson & Zeckhauser(1988)证明：人们倾向于维持现状，即使改变有明确收益。

## 陷阱 3：损失厌恶放大禀赋效应

**问题**：将卖出视为"失去"，触发损失厌恶。

❌ **Before（错误做法）**
```markdown
## 分析结论
卖掉股票感觉像失去了什么，还是持有吧。
建议：不卖。
```
*问题：禀赋效应+损失厌恶使卖出被感知为损失*

✅ **After（正确做法）**
```markdown
## 分析结论
### 框架转换
| 框架 | 感知 |
|------|------|
| 卖出=失去 | 损失厌恶触发 |
| 换仓=优化配置 | 理性决策 |

### 建议
1. 将"卖出"重新框架为"优化资产配置"
2. 关注换仓后的收益提升，而非"失去"当前股票
```
*优势：框架转换降低损失厌恶*

**原因**：损失厌恶使损失痛苦是收益快乐的2倍。重新框架可降低损失感知。
