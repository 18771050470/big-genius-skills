# 技术交付物模板

## 交付物模板 1：测试计划

```markdown
# 测试计划：[项目名称]

## 1. 测试范围
- **包含**：本次测试覆盖的功能模块
- **不包含**：明确排除的模块或场景

## 2. 测试策略
| 测试类型 | 范围 | 方法 | 负责人 |
|---------|------|------|--------|
| 功能测试 | 核心流程 | 手工+自动化 | [姓名] |
| 性能测试 | API 层 | JMeter | [姓名] |
| 安全测试 | 认证/授权 | 渗透测试 | [姓名] |

## 3. 测试环境
- **环境地址**：[URL/IP]
- **数据库**：与生产一致（脱敏）
- **浏览器**：Chrome 120+, Firefox 121+

## 4. 进度计划
| 阶段 | 开始 | 结束 | 交付物 |
|------|------|------|--------|
| 用例设计 | [日期] | [日期] | 测试用例集 |
| 测试执行 | [日期] | [日期] | 测试报告 |

## 5. 准入/准出标准
- **准入**：开发自测通过，代码冻结
- **准出**：P0/P1 缺陷清零，覆盖率 > 80%
```

## 交付物模板 4：自动化测试框架配置

```yaml
# pytest.ini
[pytest]
testpaths = tests
python_files = test_*.py
python_classes = Test*
python_functions = test_*
addopts =
    -v
    --alluredir=./allure-results
    --tb=short
    --strict-markers
markers =
    smoke: 冒烟测试
    regression: 回归测试
    slow: 慢测试

# conftest.py 示例
import pytest
from selenium import webdriver

@pytest.fixture(scope="session")
def driver():
    options = webdriver.ChromeOptions()
    options.add_argument("--headless")
    driver = webdriver.Chrome(options=options)
    yield driver
    driver.quit()
```
