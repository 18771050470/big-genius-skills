## 交付物模板 3：单篇笔记优化清单

```markdown
## 笔记优化：[笔记标题]

### 标题优化
- [ ] 包含情绪词
- [ ] 包含数字
- [ ] 包含悬念
- [ ] 字数≤20 字
- [ ] 无夸张词汇

### 封面优化
- [ ] 3:4 竖版
- [ ] 高清图片（≥720P）
- [ ] 大字标题（≥36px）
- [ ] 高对比度配色
- [ ] 文字≤8 个字

### 正文优化
- [ ] 金字塔结构
- [ ] 信息密度≥3 个知识点
- [ ] 段落清晰（每段≤3 行）
- [ ] 表情符号适度
- [ ] 引导互动（评论区提问）

### 标签优化
- [ ] 标签数量≤5 个
- [ ] 包含 1 个热点标签
- [ ] 标签与内容高度相关
- [ ] 无过气网红词

### 发布优化
- [ ] 选择最佳发布时间
- [ ] 发布前预热
- [ ] 发布后 30 分钟内回复评论
- [ ] 引导收藏和分享
```
