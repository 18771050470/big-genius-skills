## 品牌审查清单模板

```markdown
## 品牌审查清单

### 视觉检查
- [ ] Logo 使用正确（无变形/无变色）
- [ ] 色彩使用符合规范
- [ ] 字体层级正确
- [ ] 图像风格统一
- [ ] 安全空间充足

### 语言检查
- [ ] 品牌名称拼写正确
- [ ] 语气符合品牌个性
- [ ] 标语使用正确
- [ ] 无禁用词汇

### 应用检查
- [ ] 尺寸适合目标平台
- [ ] 文件格式正确
- [ ] 分辨率达标
- [ ] 无像素化/模糊

### 审查结果
- [ ] 通过
- [ ] 需修改（问题清单）
- [ ] 不通过（原因）
```
