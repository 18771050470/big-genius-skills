## Agent消息协议模板

```markdown
# [系统名] Agent消息协议

## 消息格式

### 请求消息
```json
{
  "message_id": "uuid",
  "type": "tool_call|resource_request|prompt_request",
  "sender": "agent_name",
  "receiver": "agent_name",
  "timestamp": "ISO8601",
  "payload": {
    // 请求具体内容
  },
  "context": {
    // 任务上下文
  }
}
```

### 响应消息
```json
{
  "message_id": "uuid",
  "request_id": "对应请求uuid",
  "type": "success|error|partial",
  "sender": "agent_name",
  "receiver": "agent_name",
  "timestamp": "ISO8601",
  "payload": {
    // 响应内容
  },
  "error": {
    "code": "错误码",
    "message": "错误信息",
    "suggestion": "修复建议"
  }
}
```

## 错误码定义
| 错误码 | 含义 | 处理方式 |
|--------|------|----------|
| [码] | [含义] | [处理] |

## 状态码定义
| 状态 | 含义 | 流转规则 |
|------|------|----------|
| [状态] | [含义] | [规则] |
```
