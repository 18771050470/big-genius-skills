# 动效/交互设计师技术交付物模板

## 动效规范文档模板

```markdown
## 动效规范：按钮点击反馈

### 触发条件
- 鼠标点击
- 触摸点击
- 键盘 Enter/Space

### 动画参数
| 属性 | 起始值 | 结束值 | 时长 | 缓动 |
|------|--------|--------|------|------|
| scale | 1.0 | 0.95 | 100ms | ease-out |
| scale | 0.95 | 1.0 | 150ms | ease-out |

### 状态转换
默认 → 按下（100ms）→ 释放（150ms）→ 默认

### 代码示例
```css
.button:active {
  transform: scale(0.95);
  transition: transform 100ms ease-out;
}
```

### 备注
- 禁用状态下无动画
- 支持 prefers-reduced-motion
```

## 交互原型说明模板

```markdown
## 交互原型：购物车流程

### 页面流程
商品详情 → 加入购物车 → 购物车页面 → 结算

### 交互说明
#### 加入购物车
- 触发：点击"加入购物车"按钮
- 反馈：按钮变为"已加入"，购物车图标数字 +1
- 动画：商品图片飞入购物车图标（300ms，ease-in-out）

#### 购物车页面
- 进入：从右侧滑入（400ms，ease-out）
- 返回：向右滑出（300ms，ease-in）

### 异常状态
- 库存不足：按钮禁用，显示"库存不足"
- 网络错误：Toast 提示"网络异常，请重试"
```
