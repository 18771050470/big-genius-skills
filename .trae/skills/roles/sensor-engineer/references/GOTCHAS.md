# Gotchas

> 只写 Agent 容易犯错的**非显而易见的事实**。通用知识不要写。
> 每个 Gotcha 必须包含：问题描述 + ❌ Before（错误做法） + ✅ After（正确做法） + 原因解释。

## Gotcha 1：忽略传感器输出阻抗与ADC输入阻抗匹配

**问题**：传感器输出阻抗过高，ADC采样时建立时间不足，导致测量值不准确。

❌ **Before（错误做法）**
```markdown
# 信号链设计
- 传感器输出阻抗：10kΩ
- ADC输入阻抗：1kΩ
- 直接连接，无缓冲
- 结果：ADC采样时电压跌落，测量值偏低且不稳定
```

✅ **After（正确做法）**
```markdown
# 信号链设计
- 传感器输出阻抗：10kΩ
- 增加单位增益缓冲器（运放）
- 缓冲器输出阻抗：< 1Ω
- ADC输入阻抗：1kΩ
- 结果：ADC采样稳定，测量准确

# 阻抗匹配规则
- 传感器输出阻抗 << ADC输入阻抗（至少10倍）
- 或增加电压跟随器（单位增益缓冲）
- 建立时间：τ = (R_out + R_adc) × C_sample < 采样周期/10
```

**原因**：ADC采样时需要对采样电容充电。如果传感器输出阻抗过高，充电电流不足，采样电容无法在采样时间内建立到正确电压。

## Gotcha 2：未做抗混叠滤波导致高频噪声混叠

**问题**：未在ADC前设置抗混叠滤波器，高频噪声混叠到有效频带，无法通过数字滤波去除。

❌ **Before（错误做法）**
```markdown
# 采集系统
- 信号带宽：0-100Hz
- 采样率：1kHz
- 无硬件滤波
- 结果：500Hz噪声混叠到500Hz-1kHz，再折叠到0-500Hz，污染有效信号
```

✅ **After（正确做法）**
```markdown
# 采集系统
- 信号带宽：0-100Hz
- 采样率：1kHz
- 硬件低通滤波：截止频率150Hz，-40dB/dec
- 结果：高频噪声被有效抑制，无混叠

# 抗混叠滤波设计
- 截止频率：f_c = 信号带宽 × 1.5 = 150Hz
- 衰减要求：在f_s/2 = 500Hz处衰减 > 40dB
- 滤波器阶数：2阶或更高
```

**原因**：根据采样定理，高于f_s/2的频率会混叠到有效频带。硬件抗混叠滤波必须在ADC前抑制这些高频成分，否则数字滤波无法区分混叠信号和真实信号。

## Gotcha 3：忽略参考电压的温漂

**问题**：使用普通LDO作为ADC参考电压，温度变化时参考电压漂移，导致测量误差。

❌ **Before（错误做法）**
```markdown
# 参考电压设计
- 使用普通3.3V LDO作为ADC参考
- LDO温漂：100ppm/°C
- 温度变化：20°C → 60°C（ΔT = 40°C）
- 参考电压变化：3.3V × 100ppm × 40 = 13.2mV
- 结果：12位ADC满量程误差 = 13.2mV / 3.3V × 4096 = 16 LSB
```

✅ **After（正确做法）**
```markdown
# 参考电压设计
- 使用精密参考电压源（如REF3333）
- 温漂：3ppm/°C
- 温度变化：20°C → 60°C（ΔT = 40°C）
- 参考电压变化：3.3V × 3ppm × 40 = 0.4mV
- 结果：12位ADC满量程误差 = 0.4mV / 3.3V × 4096 = 0.5 LSB

# 参考电压选型
| 类型 | 温漂 | 适用场景 |
|------|------|---------|
| 普通LDO | 100-300ppm/°C | 低精度（> 1%） |
| 精密LDO | 10-50ppm/°C | 中精度（0.1-1%） |
| 专用Vref | 1-10ppm/°C | 高精度（< 0.1%） |
```

**原因**：ADC的测量精度直接依赖于参考电压的稳定性。温度变化引起的参考电压漂移会直接影响所有测量结果，且无法通过校准消除。

## Gotcha 4：传感器与数字电路共用电源

**问题**：传感器模拟电路与MCU数字电路共用同一电源，数字开关噪声通过电源耦合到模拟信号。

❌ **Before（错误做法）**
```markdown
# 电源设计
- MCU和传感器共用3.3V电源
- 无隔离，无滤波
- 结果：MCU开关噪声（几十mV）耦合到传感器信号
- 信噪比从60dB降到40dB
```

✅ **After（正确做法）**
```markdown
# 电源设计
- MCU电源：3.3V（数字）
- 传感器电源：3.3V_A（模拟），独立LDO供电
- 滤波：传感器电源入口加10μF + 0.1μF退耦
- 隔离：磁珠或0Ω电阻分隔数字地和模拟地，单点连接
- 结果：信噪比恢复到60dB

# 电源隔离规则
- 模拟电路和数字电路分别供电
- 每个电源引脚独立退耦电容
- 数字地和模拟地单点连接（靠近ADC）
- 敏感信号远离数字开关信号
```

**原因**：数字电路的开关电流会在电源线上产生噪声。模拟电路对电源噪声敏感，共用电源会导致数字噪声污染模拟信号，降低信噪比。

## Gotcha 5：校准只做单点校准

**问题**：仅做零点校准，忽略增益误差和线性度，导致满量程附近误差大。

❌ **Before（错误做法）**
```markdown
# 校准
- 仅做零点校准：输入0 → 调整输出为0
- 无增益校准
- 结果：零点准确，但满量程误差5%
```

✅ **After（正确做法）**
```markdown
# 校准（多点校准）
- 零点校准：输入0 → 调整输出为0
- 中点校准：输入50% → 调整输出为50%
- 满点校准：输入100% → 调整输出为100%
- 线性度验证：输入25%、75%，检查误差
- 结果：全量程误差 < 0.5%

# 校准点选择
- 最少3点：0%、50%、100%
- 高精度：5点（0%、25%、50%、75%、100%）
- 校准周期：根据漂移速度，每月/每季度
```

**原因**：传感器的误差包括零点误差、增益误差和线性度误差。单点校准只能消除零点误差，增益误差和线性度误差需要通过多点校准来消除。
