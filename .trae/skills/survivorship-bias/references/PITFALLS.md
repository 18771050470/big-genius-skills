# 常见陷阱

## 陷阱 1：只研究成功者

**问题**：从成功者提取经验，忽视失败者。

❌ **Before（错误做法）**
```markdown
## 分析结论
成功企业家都坚持梦想，我们也应该这样。
建议：模仿成功者。
```
*问题：幸存者偏差使只看到成功者*

✅ **After（正确做法）**
```markdown
## 分析结论
### 完整样本分析
| 群体 | 坚持梦想 | 结果 |
|------|---------|------|
| 成功者(可见) | 是 | 成功 |
| 失败者(不可见) | 是 | 失败 |
| 失败者(不可见) | 否 | 失败 |

### 建议
1. 坚持梦想不是成功充分条件
2. 需要研究失败者为何失败
```
*优势：完整样本揭示真实规律*

**原因**：Taleb(2007)：我们只看到幸存者，看不到墓地里的失败者。成功建议可能只是幸存者特征。

## 陷阱 2：忽视沉默证据

**问题**：不可见的证据被忽视。

❌ **Before（错误做法）**
```markdown
## 分析结论
用户反馈需要这个功能，加了肯定受欢迎。
建议：立即开发。
```
*问题：幸存者偏差使忽视流失用户(沉默证据)*

✅ **After（正确做法）**
```markdown
## 分析结论
### 可见vs沉默证据
| 群体 | 可见性 | 反馈 |
|------|--------|------|
| 活跃用户 | 高 | 需要功能A |
| 流失用户 | 低(沉默) | 可能因缺少功能A离开 |

### 建议
1. 沉默证据可能更重要
2. 调研流失用户
```
*优势：识别沉默证据*

**原因**：沉默证据是不可见的失败案例，但可能包含关键信息。

## 陷阱 3：成功归因错误

**问题**：将幸存者特征归因为成功原因。

❌ **Before（错误做法）**
```markdown
## 分析结论
成功公司都有开放式办公室，我们也应该有。
建议：改造办公室。
```
*问题：幸存者偏差使相关性被误认为因果*

✅ **After（正确做法）**
```markdown
## 分析结论
### 因果vs相关
| 特征 | 成功公司 | 失败公司 | 因果性 |
|------|---------|---------|--------|
| 开放办公室 | 60% | 50% | 低 |
| 产品市场匹配 | 90% | 10% | 高 |

### 建议
1. 开放办公室与成功相关性弱
2. 聚焦高因果性因素
```
*优势：区分相关和因果*

**原因**：幸存者特征可能是巧合，非成功原因。需要对比成功和失败群体。
