- 内层L3走普通信号，避免高速信号干扰

## 交付物模板 2：布局设计说明

```markdown
# 布局设计说明：[项目名称]

## 布局原则
- **信号流向**：左→右（输入→处理→输出）
- **热管理**：大功率器件靠近板边，热敏感器件远离热源
- **可测试性**：关键信号预留测试点，调试接口 accessible

## 关键器件布局

| 器件 | 位置 | 理由 | 关键尺寸 |
|------|------|------|---------|
| MCU | 板子中心 | 信号辐射最小化 | - |
| 晶振 | MCU下方 | 走线最短 | 距离 < 20mm |
| DCDC | 电源入口 | 减少电源走线长度 | - |
| USB接口 | 板边 | 方便插拔 | - |
| 传感器 | 远离DCDC | 避免电源噪声 | 距离 > 30mm |

## 布局检查清单
- [ ] 晶振到MCU距离 < 20mm
- [ ] 去耦电容到电源引脚 < 2mm
- [ ] 高速信号连接器靠近板边
- [ ] 测试点不被器件遮挡
- [ ] 器件间距 ≥ 0.5mm
```

## 交付物模板 3：布线规则表

```markdown
# 布线规则：[项目名称]

## 通用规则

| 参数 | 值 | 说明 |
|------|-----|------|
| 最小线宽 | 0.1mm | 普通信号 |
| 最小间距 | 0.1mm | 普通信号 |
| 最小过孔 | 0.3mm/0.15mm | 孔径/焊盘 |
| 锐角 | 禁止 | 用45°或圆弧 |

## 高速信号规则

| 信号 | 阻抗 | 线宽 | 间距 | 等长误差 | 过孔限制 |
|------|------|------|------|---------|---------|
| USB D+/D- | 90Ω 差分 | 0.15mm | 0.15mm | < 50mil | 最多2个 |
| HDMI TMDS | 100Ω 差分 | 0.12mm | 0.18mm | < 20mil | 最多2个 |
| Ethernet | 100Ω 差分 | 0.15mm | 0.15mm | < 50mil | 最多2个 |
| DDR CLK | 差分 | - | - | < 10mil | 最多2个 |

## 电源规则

| 电源 | 线宽 | 过孔 | 铺铜 |
|------|------|------|------|
| 3.3V | ≥ 0.5mm | 2个并联 | 是 |
| 1.8V | ≥ 0.3mm | 1个 | 是 |
| GND | - | 密集 | 完整平面 |

## 特殊规则
- 高速信号下方必须有完整参考平面
- 差分对必须对称布线，避免分开走线
- 时钟信号远离IO和电源
- 模拟信号与数字信号分开布线
```
