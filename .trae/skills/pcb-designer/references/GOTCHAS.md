# Gotchas

> 只写 Agent 容易犯错的**非显而易见的事实**。通用知识不要写。
> 每个 Gotcha 必须包含：问题描述 + ❌ Before（错误做法） + ✅ After（正确做法） + 原因解释。

## Gotcha 1：差分对不等长导致时序偏移

**问题**：USB、HDMI等差分对的P/N线长度差过大，导致信号到达时间不同步，眼图闭合。

❌ **Before（错误做法）**
```markdown
# USB 差分对布线
- D+ 长度：1200mil
- D- 长度：1050mil
- 长度差：150mil（远超50mil限制）
- 结果：USB 2.0识别不稳定，大文件传输报错
```

✅ **After（正确做法）**
```markdown
# USB 差分对布线
- D+ 长度：1200mil
- D- 长度：1180mil（绕线补偿）
- 长度差：20mil（< 50mil限制）
- 结果：USB稳定运行，通过眼图测试

# 等长补偿方法
- 在较短的那根线上绕蛇形线
- 绕线幅度：3W（W为线宽）
- 绕线间距：≥ 2W
```

**原因**：差分信号的抗干扰能力依赖于P/N线的对称性。长度差会导致共模噪声增加、时序偏移，高速时眼图闭合。

## Gotcha 2：高速信号跨分割导致EMI恶化

**问题**：高速信号走过地平面分割处，回流路径断裂，导致辐射发射超标。

❌ **Before（错误做法）**
```markdown
# 高速信号布线
- 信号层：Top
- 参考层：GND（但被分割为数字地和模拟地）
- 信号从数字区域走到模拟区域，跨过地分割
- 结果：EMI测试辐射超标10dB
```

✅ **After（正确做法）**
```markdown
# 高速信号布线
- 信号层：Top
- 参考层：完整GND（不分割）
- 数字地和模拟地在单点连接（靠近电源入口）
- 高速信号始终在完整GND上方
- 结果：EMI测试通过

# 跨分割处理
- 如果必须跨分割，在跨接处放置 stitching via（缝合过孔）
- 或使用桥接电容连接两侧地平面
```

**原因**：高速信号的回流电流在参考平面上紧随信号路径。跨分割会迫使回流电流绕远路，形成环路天线，增加辐射。

## Gotcha 3：去耦电容放太远失去作用

**问题**：去耦电容距离电源引脚太远，寄生电感过大，高频去耦效果丧失。

❌ **Before（错误做法）**
```markdown
# 去耦电容布局
- MCU电源引脚在板子左侧
- 去耦电容放在板子右侧
- 距离：30mm
- 结果：电源噪声大，MCU复位不稳定
```

✅ **After（正确做法）**
```markdown
# 去耦电容布局
- 每个电源引脚旁边放置0.1uF电容
- 距离：≤ 2mm
- 电容地焊盘直接打过孔到地平面
- 结果：电源噪声 < 50mV，系统稳定

# 去耦电容布局规则
- 小电容（0.1uF）靠近引脚（< 2mm）
- 大电容（10uF）稍远（< 10mm）
- 电容地焊盘直接打地过孔，不经过走线
```

**原因**：去耦电容的寄生电感（焊盘+过孔+走线）会限制高频响应。距离越远，寄生电感越大，去耦效果越差。

## Gotcha 4：忽视PCB板材对阻抗的影响

**问题**：阻抗计算时使用错误的介电常数，导致实际阻抗与目标偏差大。

❌ **Before（错误做法）**
```markdown
# 阻抗计算
- 使用FR4默认介电常数4.5
- 实际板材介电常数4.2
- 目标阻抗：90Ω
- 实际阻抗：82Ω（偏差9%）
- 结果：USB眼图测试失败
```

✅ **After（正确做法）**
```markdown
# 阻抗计算
- 向板材供应商确认介电常数（DK）和损耗因子（DF）
- 使用实际DK值计算线宽
- 要求供应商提供阻抗测试报告
- 目标阻抗：90Ω ± 10%
- 实际阻抗：88Ω（偏差2%）
- 结果：通过眼图测试

# 关键参数
| 板材 | DK@1GHz | 适用场景 |
|------|---------|---------|
| 普通FR4 | 4.2-4.5 | 一般信号 |
| 高速FR4 | 3.8-4.0 | > 1GHz |
| Rogers | 3.0-3.5 | RF/微波 |
```

**原因**：不同板材的介电常数差异可达10%，直接影响阻抗计算。必须使用实际板材参数，并在生产前要求阻抗测试。

## Gotcha 5：忘记考虑生产公差

**问题**：设计时按理想值计算，未考虑生产公差，导致批量生产时良率低。

❌ **Before（错误做法）**
```markdown
# 设计规则
- 线宽：0.1mm（理想值）
- 间距：0.1mm（理想值）
- 未考虑公差
- 结果：批量生产时10%的板子短路
```

✅ **After（正确做法）**
```markdown
# 设计规则（含公差）
- 线宽：0.1mm + 0.02mm公差 = 设计0.12mm
- 间距：0.1mm + 0.02mm公差 = 设计0.12mm
- 与制造商确认工艺能力：±0.02mm
- 结果：批量生产良率 > 95%

# 公差考虑清单
- [ ] 线宽公差（通常 ±10-20%）
- [ ] 孔径公差（通常 ±0.05mm）
- [ ] 层偏公差（通常 ±0.05mm）
- [ ] 阻抗公差（通常 ±10%）
```

**原因**：PCB生产有固有公差（蚀刻、钻孔、层压）。设计时必须考虑公差，确保在最差情况下仍满足要求。
