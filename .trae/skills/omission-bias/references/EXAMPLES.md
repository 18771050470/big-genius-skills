# 分析示例

## 示例 1：遗漏偏差——医疗决策

**输入**：我不给孩子打疫苗，因为听说有副作用，不打至少不会主动伤害他。

```
Step 1: 识别遗漏信号 → 产出：发现"不打至少不会主动伤害"表述，认为不作为比作为道德上更好
Step 2: 评估行为vs遗漏 → 产出：不打疫苗(遗漏)导致疾病风险10%，打疫苗(行为)副作用风险0.001%
Step 3: 分析遗漏机制 → 产出：行为责任：行为导致的伤害比遗漏导致的伤害感觉更严重；后悔预期：行为错误比遗漏错误更后悔
Step 4: 校正遗漏偏差 → 产出：建议比较实际风险，而非行为vs遗漏的道德差异
```

**输出**：
```
## 分析结论
### 核心发现
1. 不打疫苗(遗漏)疾病风险10%，打疫苗(行为)副作用风险0.001%
2. 遗漏偏差使"不作为"被感知为更安全，但实际风险更高
3. Ritov & Baron(1990)实验：人们宁愿孩子因不打疫苗而死，也不愿因疫苗副作用而死

### 建议（按优先级）
1. [高优先级] 比较实际风险：疾病风险10% >> 副作用风险0.001%
2. [高优先级] 理解遗漏偏差：不作为≠无风险
3. [中优先级] 咨询专业医生，基于数据决策

### 关键假设
- 风险数据来自可靠来源
- 疫苗有效性高

### 风险
- 遗漏偏差可能导致高风险的不作为
- 需要区分道德感知和实际风险
```

## 示例 2：遗漏偏差——投资决策

**输入**：我没买那只股票，虽然错过了10倍收益，但至少没亏钱，比买了亏了强。

```
Step 1: 识别遗漏信号 → 产出：发现"没买...至少没亏"表述，认为不行动的错过比行动的损失更好接受
Step 2: 评估行为vs遗漏 → 产出：没买(遗漏)错过收益，买了(行为)可能亏损，但遗漏的机会成本真实存在
Step 3: 分析遗漏机制 → 产出：行为责任：行动导致的损失更痛苦；反事实思维：行动后亏损更容易想象"如果不行动"
Step 4: 校正遗漏偏差 → 产出：建议用机会成本评估遗漏，而非道德感知
```

**输出**：
```
## 分析结论
### 核心发现
1. 没买(遗漏)机会成本：错过10倍收益
2. 遗漏偏差使"没亏"被感知为更好，但机会成本真实存在
3. 行为损失和遗漏损失在经济上等价，但心理感受不同

### 建议（按优先级）
1. [高优先级] 用机会成本评估遗漏：错过的收益=实际损失
2. [高优先级] 建立投资决策框架，而非回避行动
3. [中优先级] 理解行为损失和遗漏损失心理差异

### 关键假设
- 机会成本可量化
- 投资决策框架可降低偏差

### 风险
- 遗漏偏差可能导致过度保守
- 需要平衡风险和机会
```

## 反例：不该触发此 Skill 的场景

**输入**：这个手术风险太高，成功率只有10%，选择不手术是合理的。

**原因**：这是基于风险数据的理性决策(10%成功率)，不是遗漏偏差。遗漏偏差指因"不作为"而忽视实际风险。
