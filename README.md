# Big Genius Skills

> 🧠 面向创业者的全栈 AI Agent 技能体系 —— 从商业战略到代码实现的完整闭环。

## 核心架构

采用 **四层分层架构**：

| 层级 | 职责 | 规模 |
|------|------|------|
| **L0 Commander** | 意图识别、任务分解、技能编排 | 1 个 |
| **L1 思维层** | 356 个思维模型（因果/系统/逆向/概率等） | 356 个 |
| **L2 角色层** | 132 个专业角色（架构师/工程师/顾问等） | 132 个 |
| **L3 工具层** | 文档/数据/浏览器/安全等专用工具 | 8+ 个 |

**总计 497+ 个技能单元**，覆盖商业分析、全栈开发、AI 集成、嵌入式、合规风控等全链路场景。

## 快速开始

```bash
# 克隆项目
git clone https://github.com/yourusername/big-genius-skills.git

# 查看技能目录
ls skills/
# ├── commander/          # 总指挥
# ├── thinking/           # 思维模型
# ├── roles/              # 专业角色
# └── tool/               # 工具技能
```

## 设计理念

- **分层解耦**：每层职责清晰，单向调用，禁止循环依赖
- **触发精准**：每个 Skill 的 description 包含明确触发信号
- **验证闭环**：每步执行后有量化验证标准
- **简洁优先**：核心指令聚焦，详情按需加载

## 使用示例

**快速路径**（简单任务，≤3 步）：
```
用户：帮我写个斐波那契函数
Agent：直接输出代码
```

**标准路径**（复杂任务，>3 步）：
```
用户：设计一个电商订单模块
Agent：Commander → 系统思维 → 后端架构师 → QA 工程师
```

## 核心文件

| 文件 | 说明 |
|------|------|
| `AGENT_CORE_RULES_v2.txt` | 全局执行规则（简洁优先、验证闭环、范围纪律） |

| `skills/` | 技能目录（按层级组织） |
